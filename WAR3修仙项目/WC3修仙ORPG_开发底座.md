# WC3 修仙 ORPG · 开发底座 v4.1（合并 v2.5 系统详细设计）

> **唯一总纲。** 本文档合并了 v4.0 的版本路线框架与 v2.5 的系统详细设计。v4.0 贡献了版本→系统映射表和 V0-V7 优先级体系；v2.5 贡献了 A~F 六章详细系统描述。系统编码以第一部分"A~AS"为准，第四部分 A1~F 为历史编码，内容仍有效。

## 第一部分：项目总纲

### 0\. 文档说明

* 本文档用途：项目唯一总纲，用于策划、程序、美术、测试统一认知。
* 本文档不写完整数值表、不写具体剧情文本、不替代系统详细设计。
* 文档结构：总纲 → 版本路线 → 系统映射 → 系统详情 → 附录索引。

### 1\. 引擎与项目形态

* 引擎：Warcraft III / WC3 地图开发环境。
* 平台：KK 对战平台。
* 形态：4-5 人房间 ORPG + 服务器存档。
* 主体验：个人单机刷怪成长为主。
* 辅体验：房间社交、组队协作、拍卖行、全服经济。
* 类型定位：暗黑3/4式刷怪掉宝 + 修仙境界成长 + 长线 ORPG 存档。

### 2\. 整体设计思想

* 核心一句话：**修为靠闭关慢慢长（修炼相关系统），命运靠出门刷出来（战斗相关系统）。**
* 单人闭环优先：不开房、不交易、不组队，也必须能完整成长。
* 房间是加味：多人提高热闹、效率、合作事件，但不能成为进度门槛。
* 全服经济是流通层：拍卖行不能替代刷怪掉宝。
* 境界是底子：决定基础数值、地图权限、压制关系。
* 装备是上限：决定同境界内强弱、Build、刷宝追求。
* 刷怪核心：打怪 → 掉落 → 吃丹/炼丹/装备 → 变强 → 挑更高难度。
* 修仙核心：打坐/闭关 → 积累修为 → 收集突破材料 → 渡劫/心魔 → 境界跃迁。
* 三层世界：个人存档世界、房间实例世界、全服经济世界。

### 核心设计原则（继承自 v2.5）

#### 原则一：境界 vs 装备的分工

> 境界 = 容器（潜力下限），装备 = 内容物（变现上限）。两套东西尽量不重叠，让玩家有明确认知。

| 维度 | 境界给（下限/潜力） | 装备给（上限/变现） |
|---|---|---|
| 攻击/防御/生命 | **基础基数**，大境界数量级跃迁 | 加成百分比 + 固定值加成 |
| 暴击/攻速/施法 | ❌ 不给 | 暴击率/暴伤/攻速/冷却，纯装备区 |
| 特殊机制 | 境界压制（PvE/PvP系数） | 吸血/反弹/护盾/触发/破压制 |
| 技能 | 技能/神通等级解锁上限 | 技能伤害加成/冷却缩减 |
| 修炼 | 打坐效率基数 | 灵气加成/修炼加速词条 |
| 区域权限 | 解锁新地图 | ❌ 不给 |

**数值模型**：`实际属性 = 境界基数 × (1+装备加成%) + 装备固定值`

**关键效果**：
- 大境界碾压靠下限：筑基裸装 > 练气毕业装
- 同境界内卷靠上限：同是筑基，装备拉开3-5倍
- 偷渡靠上限搏下限：练气毕业装 vs 筑基垃圾装有得打

#### 原则一附：小境界突破的收获

小境界不直接给大量属性基数（那是装备的事），而是**横向打开容器**：

| 小阶突破 | 游戏实质 | 修仙叙事包装 |
|---|---|---|
| 提升法力上限 | 蓝量/能量池扩容 | "经脉拓宽，法力如河" |
| 神识增强 | 解锁更多法宝出战位 | "神识外放，三丈之内尽在掌握" |
| 术法品阶提升 | 技能可升到更高等级 | "此式以练气之力是微风，以金丹之力是风暴" |
| 体质淬炼 | 少量生命/防御下限 | "每破一阶，凡胎脱去一分" |
| 威压范围 | 对低小阶敌压制系数微增 | "高你一阶，便压你一头" |
| 丹火/器火升温 | 炼丹炼器成功率/品质加成 | "金丹之火可炼灵器，元婴之火可炼法宝" |

#### 原则一附：成长型装备（本命法宝）

- 只跨**大境界**成长，不跨小境界（防泛滥）
- 一个角色同时只能装备1-2件本命法宝
- 极稀有来源：独有BOSS掉落/轮回底蕴/特殊机缘
- 叙事锚点："练气期炼的剑，陪你走到飞升"

#### 原则二：三层世界边界

> 不是"单机+MMO二选一"，而是三层各司其职。

| 层面 | 负责什么 | 绝不能做的事 |
|---|---|---|
| **个人存档世界** | 境界/灵根/命格/洞府/功法/法宝/修仙录/突破主药/独有BOSS | 不能通过交易/组队绕过 |
| **房间实例世界** | 普通怪掉落/高频BOSS/简单机缘（组队共享）；怪物强度随人数适配 | 不能让组队变成进度门槛 |
| **全服经济世界** | 拍卖行——只流通稀有品：功法密卷/特殊材料/极品法宝装备 | 不能让"买装备"替代"刷装备" |

**铁律**：玩家不开房、不交易、不组队，也必须能完整刷怪、掉宝、突破、成长。房间是加味，不是门槛。

#### 原则三：难度层级

| 层级 | 机制 | 说明 |
|---|---|---|
| **空间难度** | 不同地图不同境界要求 | F7 高级区偷渡，允许低境界用操作/符箓/法宝进入 |
| **纵向难度** | 同一区域可调魔化等级（1-5层） | 怪物属性放大+掉落品阶提高+奖励倍率，读取新怪物配置 |
| **房间难度** | 4-5人开房→怪物血量/伤害适度加强→掉落数量/品质小幅提升 | 组队"划算但不必须" |
| **秘境难度** | 蓝门/屠魔录分层（1-N层） | 层数×怪物系数，独立难度曲线 |

## 第二部分：版本路线

### V0 可运行版

核心体验：**玩家能进房、创建角色、打怪、获得物品、保存数据。**  
目标不是好玩，WC3 + KK + 存档 + 基础战斗链路可跑通。

### V1 30秒首爽版

核心体验：**进房30秒内打死怪，看到掉落，吃丹或拾取灵气，立刻变强。**  
这是项目第一生命线。

### V2 首次突破版

核心体验：**玩家完成一次“修炼→攒材料→突破→特效播报”的修仙跃迁。**  
这一版要让玩家明确感到：这不是传奇换皮，是修仙。

### V3 刷宝闭环版

核心体验：**刷怪、爆装、小极品、词缀、偷渡、高难区域形成暗黑式循环。**  
这一版决定能不能长期刷。

### V4 长线养成版

核心体验：**洞府、功法、法宝、宗门、图鉴，让角色像在另一个生活里成长。**  
这一版负责留存。

### V5 社交经济版

核心体验：**房间协作、交易、拍卖、播报、排行榜，让玩家感到自己在一个全服生态里。**  
这一版负责社交和外部比较。

### V6 终局轮回版

核心体验：**飞升、轮回、重修，给长线玩家一个“再来一世但底蕴更厚”的理由。**

# 第十七章 已决策项（v2.2 更新）

| # | 决策项 | 决定 |
|---|---|---|
| 1 | 核心进阶曲线 | 境界=容器（底线/潜力），装备=内容物（变现/上限）。大境界碾压靠下限，同境界内卷靠上限。公式：`境界基数×(1+装备%)+固定值` |
| 2 | 小境界收获 | 横向打开容器（法力扩容/神识→法宝位/术法品阶/体质/威压/丹火），不给大量属性基数 |
| 3 | 成长型装备 | 只跨大境界，单角色1-2件，极稀有 |
| 4 | 三层世界边界 | 个人存档=一切基础；房间共享=普通怪掉落+高频BOSS+简单机缘；全服经济=拍卖行只流通稀有品 |
| 5 | 移动/飞行 | 战斗场景步行。大世界独立场景（云雾+小角色+御剑）|
| 6 | 坐骑 | 要，战斗和大世界都要，含功能性 |
| 7 | 召唤/家奴/灵兽 | 分立。战斗助手早期做；家奴后置P3 |
| 8 | 难度层级 | 空间+纵向（魔化1-5层）+房间人数加强+秘境分层，四层 |
| 9 | 怪物词缀 | 要，精英挂1-2词缀，WC3特效绑定，视觉强区分 |
| 10 | 房间怪物加强 | 人数→怪物强度+掉落适配；难度层→读取新配置表 |
| 11 | 装备槽位 | 暂不锁定，详设确定 |
| 12 | 套装 | 要做，数量后定 |
| 13 | 商业化 | P2-P3占位。KK商城抽奖+简单直购 |
| 14 | 活跃 | 修士一日清单式，P0不做后置P1 |
| 15 | 声望 | 宗门声望+区域声望，作为游戏玩法内置于宗门晋升和区域解锁 |
| 16 | 主线 | 骨架：入道引导+筑基任务，5-10个 |
| 17 | 强化/回收 | P0简化：掉落→使用→回收；P1再加淬炼/拆解 |
| 18 | 轮回=赛季 | 远期可行，思路记录。普通+赛季双模式。当前P3，P0只预留字段 |
| 19 | 坊市 | 要，核心=价格浮动+鉴别机制+捡漏+全买必亏 |
| 20 | 阵法 | 要，Buff机制实现，三类阵法（洞府/战斗/秘境），阵旗阵盘由符箓系统产出 |
