## 项目概要

WC3 引擎 / KK 对战平台，4-5 人房间 + 服务器存档的修仙 ORPG。底子是传奇式刷怪掉宝，外面包一层完整的修仙生活模拟。

核心体验：每天都是修士的一日——上线收闭关成果，出门历练刷材料，回家炼丹突破，睡前挂上闭关。不是"进图就干、干完就走"的传统 WC3 刷子图，也不是"一局速通"的快餐修仙。

> 修为靠闭关慢慢长，命运靠出门刷出来。

## 项目大前提

- **平台**：魔兽争霸 3 引擎，KK 对战平台（魔兽 RPG 地图平台）
- **玩法**：ORPG，4-5 人 / 房，服务器存档，非一局重开。本质是长线单机 + 偶尔开房共存的松散社交
- **原始底子**：复古传奇手感——刷怪→掉装备→升级（技能 / 铭文 / 词条系统已基本完成）
- **题材转变**：因 IP 版权问题，从传奇转为修仙仙侠
- **当前状态**：设计方法论已收敛至 v4.0 开发总纲（8 段版本规划 + 45 系统 LIST + 版本映射表），进入 P0 详细设计阶段
- **核心约束**：WC3 引擎技术限制（同屏单位数上限、网络同步、存档结构）；KK 平台用户以中年怀旧玩家为主，对强 PvP 兴趣有限
- **核心差异化**：不吃丹修仙的"一局速通"，也不做纯放置挂机——用"修士一日"的生活节奏串起刷怪爽感和修仙生活感

## 项目背景

项目始于传奇题材的 ORPG 刷宝图，底子已有完整的刷怪→掉装备→升级循环（技能 / 铭文 / 词条基本完成）。因 IP 问题被迫转题材，选定修仙仙侠方向。

双线并行设计：底层保留传奇式的刷怪掉宝核心循环（打怪→掉宝→变强→打更强的怪→掉更好的宝）；上层叠加修仙特有的系统层（修炼 / 打坐 / 境界突破 / 宗门 / 洞府 / 机缘 / 轮回）。

参考竞品与灵感：和平精英"吃丹修仙"（极简修仙闭环、零门槛爽感验证）、鬼谷八荒（NPC 生态与境界系统）、觅长生（炼丹深度与开放式成长）、一念逍遥（放置修仙的反面教训）。差异化核心：不是纯刷子图换皮，也不是纯放置挂机——用"修士一日"的生活节奏串起两者。

## 已确认结论

### 核心设计哲学（2026-05-09 最终校准）

**一句话定性**（决定）：

> 修为靠闭关慢慢长，命运靠出门刷出来。

**双轴模型**（决定）：闭关和刷怪不在同一条收益线上，不能用"修为占比"去判断会不会削弱刷怪。它们是两条平行轴：

| | 闭关轴（修仙时间轴） | 刷怪轴（命运事件轴） |
|---|---|---|
| **回答的问题** | "我每天都在修行" | "今天发生了什么" |
| **核心产出** | 修为持续上涨，境界逼近瓶颈 | 丹材、突破主药、法宝、装备、功法残页、灵石、蓝门触发 |
| **节奏** | 缓慢、可预期、不退 | 脉冲式、"叮"、不可预测 |
| **惊喜** | **不给。** 不掉极品装备、法宝、独有材料、蓝门、唯一 BOSS 奖励 | **大量。** 爆丹材、爆法宝、遇机缘、偷渡成功、拿突破钥匙 |

**关键约束**（决定）：
- 闭关 / 打坐占修为 60-70% 没有问题——前提是闭关不给核心惊喜
- 刷怪不需要和闭关比经验效率。刷怪的价值是：突破材料、财富、装备、法宝、机缘、交易品、故事
- 杀怪不直接给修为，靠"炼丹→服食"中间环节转化
- 每日丹药上限防脚本
- **突破必须同时吃修为和材料（主药来自刷怪）**，否则刷怪失去必要性

**丹药递减曲线**（决定）：
- 练气-筑基期：丹药占修为进度 10-30%（确保"30 秒首爽"）
- 金丹-元婴期：丹药占比降到 5-10%，玩家开始更依赖打坐
- 化神-渡劫期：普通丹 < 1%，完全回归修炼主路径

**修士一日节奏模型**（决定）：

```
上线 → 看闭关收益（修为涨了多少，离突破还差多少）
     → 根据当前瓶颈决定出门刷什么：
          缺突破主药 → 刷妖兽
          缺钱 → 刷材料卖拍卖行
          想搏命 → 偷渡高级区
          想赌机缘 → 刷蓝门或夜间魔物
     → 回家整理（服丹/炼丹/炼器/整理法宝）
     → 尝试突破
     → 继续闭关，等明天
```

不是"刷到累"，而是"闭关有进度，出门有故事"。

**境界 vs 装备的分工模型**（决定 · 2026-05-12 新增）：

> 境界 = 容器（潜力下限），装备 = 内容物（变现上限）。两套东西不重叠。

**数值模型**（决定）：`实际属性 = 境界基数 × (1+装备加成%) + 装备固定值`

| 维度 | 境界给（下限/潜力） | 装备给（上限/变现） |
|---|---|---|
| 攻击/防御/生命 | 基础基数，大境界数量级跃迁 | 加成百分比 + 固定值加成 |
| 暴击/攻速/施法 | ❌ 不给 | 暴击率/暴伤/攻速/冷却，纯装备区 |
| 特殊机制 | 境界压制（PvE/PvP系数） | 吸血/反弹/护盾/触发/破压制 |
| 技能 | 技能/神通等级解锁上限 | 技能伤害加成/冷却缩减 |
| 修炼 | 打坐效率基数 | 灵气加成/修炼加速词条 |
| 区域权限 | 解锁新地图 | ❌ 不给 |

**关键效果**（决定）：
- 大境界碾压靠下限：筑基裸装 > 练气毕业装
- 同境界内卷靠上限：同是筑基，装备拉开3-5倍
- 偷渡靠上限搏下限：练气毕业装 vs 筑基垃圾装有得打

**小境界突破的新模型**（决定 · 2026-05-12 新增）：
小境界不直接给大量属性基数（那是装备的事），而是**横向打开容器**：

| 小阶突破 | 游戏实质 | 修仙叙事包装 |
|---|---|---|
| 提升法力上限 | 蓝量/能量池扩容 | "经脉拓宽，法力如河" |
| 神识增强 | 解锁更多法宝出战位 | "神识外放，三丈之内尽在掌握" |
| 术法品阶提升 | 技能可升到更高等级 | "此式以练气之力是微风，以金丹之力是风暴" |
| 体质淬炼 | 少量生命/防御下限 | "每破一阶，凡胎脱去一分" |
| 威压范围 | 对低小阶敌压制系数微增 | "高你一阶，便压你一头" |
| 丹火/器火升温 | 炼丹炼器成功率/品质加成 | "金丹之火可炼灵器，元婴之火可炼法宝" |

**成长型装备（本命法宝）**（决定 · 2026-05-12 新增）：
- 只跨大境界成长，不跨小境界（防泛滥）
- 单角色限制1-2件
- 极稀有来源：独有BOSS/轮回底蕴/特殊机缘

**三层世界边界**（决定 · 2026-05-12 新增）：

| 层面 | 负责什么 | 绝不能做的事 |
|---|---|---|
| 个人存档世界 | 境界/灵根/命格/洞府/功法/法宝/修仙录/突破主药/独有BOSS | 不能通过交易/组队绕过 |
| 房间实例世界 | 普通怪掉落/高频BOSS/简单机缘（组队共享） | 不能让组队变成进度门槛 |
| 全服经济世界 | 拍卖行只流通稀有品 | 不能让"买装备"替代"刷装备" |

铁律：玩家不开房、不交易、不组队，必须能完整刷怪、掉宝、突破、成长。P0 全为单人体验，房间后移到 P1。

**难度层级四层模型**（决定 · 2026-05-12 新增）：

| 层级 | 机制 |
|---|---|
| 空间难度 | 不同地图不同境界要求，允许低境界偷渡 |
| 纵向难度 | 同一区域可调魔化等级1-5层，读取新怪物配置 |
| 房间难度 | 4-5人开房→怪物血量/伤害加强→掉落适配（组队"划算但不必须"）|
| 秘境难度 | 蓝门/屠魔录分层（1-N层）×怪物系数 |

**怪物词缀**（决定 · 2026-05-12 新增）：精英怪随机挂1-2个词缀，WC3单位特效锚点绑定实现强视觉区分，词缀影响掉落品质。

**阵法系统**（决定 · 2026-05-12 新增）：三类阵法——洞府阵法（聚灵/护府）、战斗阵法（困/陷阱/迷，用Buff实现）、秘境阵法（古传送/禁制）。阵旗阵盘由符箓系统产出。修仙小说弱打强的核心手段。

**坊市系统**（决定 · 2026-05-12 新增）：区别于商店明码标价。NPC随机摊位、价格浮动±50%、丹道/器道等级达标可鉴别"被低估"品、全买必亏防刷。

**道器系统**（决定 · 2026-05-12 新增）：每人一种生涯级被动效果，不可交易不可更换，轮回重置重新随机。灵感来源：修仙小说中独一无二的先天奇物。

**图鉴系统**（决定 · 2026-05-12 修订）：从原有"装备图鉴"改为修仙六类收集——丹方集、药材谱、奇珍录、怪物志、功法阁。奖励不限于属性（隐藏丹方/种子礼包/特殊材料/称号等）。

**版本规划**（决定 · 2026-05-13 更新）：从原 P0-P4 五段改为 0-7 八段等长递进——内部技术验收版/战斗首爽版/修炼突破版/暗黑刷宝版/生产消化版/长线生活版/飞升轮回版/全服生态版。详见 v4.0。

**功法决定修炼方式**（决定）：
- 修为只来自打坐（在线/离线），功法各管各的方向
- 修炼功法 → 修炼速度加成（B 类）
- 剑修 → 打坐 + 每日斩剑 N 次（前置动作门槛，战斗属性加成）
- 阵修 → 打坐 + 阵法参悟/神识探测（阵法/神识属性加成）
- 兽修 → 打坐 + 灵兽滋养/御兽（灵兽属性加成）
- 丹修 → 炼丹过程中也增长修为（第二个修为来源，不走速度公式，挂炼丹动作）
- 魔修 → 需特殊材料催化，效率高
- 不做"每个功法都加修为"——与长老服务体系一致，build 自由组合

## 重要讨论

### 2026-05-07：题材本质与设计定位
- **决定**：从传奇题材转为修仙仙侠题材（版权IP原因）
- **事实**：KK平台（魔兽RPG地图平台），受众最广的ORPG类型为魔兽世界观和东方玄幻
- **判断**：修仙体验核心 = 境界突破（阶梯式质变而非线性升级）+ 机缘（条件发现而非概率掉落）+ 功法即身份 + 慢节奏孤独氛围
- **判断**：WC3 ORPG = 有存档的持续单人成长 + 可开房多人松散共存，契合修仙"散修+宗门"若即若离结构
- **判断**：参考一念逍遥的反面教训——突破不能是数值膨胀，必须是玩法横向打开（新能力、新区域、新机制）

### 2026-05-08：设计策划会最终收束

**P0 - 核心修仙爽感闭环（MVP，3-6 个月）**：

| 系统 | 核心要点 |
|------|----------|
| 修炼 / 闭关 | 修为主来源。打坐主动获取，闭关离线结算。洞府、宗门修炼地、灵气峰值都是该系统的配置调用。闭关不给核心惊喜 |
| 刷怪掉落 | 不承担主要修为。产出丹材、突破主药、法宝材料、装备、功法残页、符箓材料、灵石、蓝门触发。高级区=掉落品阶更高+PvE 压制放水 |
| 炼丹 / 吃丹 | 前期自动炼普通丹，吃丹立刻涨修为。后期手动炼丹出极品，可出售、突破、战斗前使用。**每日丹药上限** |
| 境界突破 | 凡人→练气→筑基→金丹→元婴→化神→渡劫→飞升。修为满只是条件之一，还需要主药/材料/突破事件（心魔/雷劫）。失败不卡死（保底必成）。突破主药必须刷怪获取（强制刷怪点） |
| 境界压制 | 重点服务 PvE 日常。高境界打低境界怪明显轻松。PvP 只做简单公式修正（系数 0.5），不早期复杂化。法宝/极品丹/功法/机缘可破压制 |
| 高级区偷渡 | 高级区不锁死。低境界可靠路线、符箓、装备、操作偷资源。奖励必须足够大：跨境丹材、法宝材料、高价资源。死亡惩罚有但不过重 |

**P1 - 身份与留存（6-9 个月）**：

| 系统 | 核心要点 |
|------|----------|
| 洞府 / 灵气 | 洞府周期积累灵气，每天有高峰时段后衰减。可放灵植、法宝、灵兽提升闭关或炼丹收益 |
| 功法 / 法宝 | 技能和词条修仙化。功法决定 build，法宝替代核心装备，提供战斗力、破压制、特殊机制 |
| 宗门 / 派系 | 宗门是修炼方向。丹堂炼丹赚钱，器堂炼器制符，剑堂战斗强，阵堂阵法神识，兽堂灵兽驾驭。先做少量清晰流派 |
| 日夜 / 时段 | 清晨灵气高，夜晚魔物刷新，深夜稀有怪概率提高。灵植、怪物、事件与时间挂钩 |
| 基础拍卖行 | 先做单机 NPC 拍卖，保证经济流通。极品丹、法宝材料、符箓成为交易核心。全服拍卖后置 |

**P2 - 深度与差异化（9-12 个月）**：

| 系统 | 核心要点 |
|------|----------|
| 机缘系统 | 暗黑 3 蓝门式副本 + 随机洞府 + 特殊事件。奖励稀有材料、功法、法宝、突破推动 |
| 一次性稀有 BOSS | 按玩家存档维度唯一。刷到并击杀后永久标记，不再刷新，掉极珍贵奖励 |
| 屠魔录 | 魔界周期侵蚀人间，开启大秘境分层挑战和排名。承担中后期刷榜目标 |
| 修仙录 / 道争录 | 自动记录突破、机缘、击杀、交易、NPC 关系。生成战报和玩家个人江湖史（传播命脉） |

**P3 - 远期（12 个月+）**：

| 系统 | 核心要点 |
|------|----------|
| NPC 生态 / 家奴 | NPC 友好度、传闻、可收服魔性 NPC。家奴可派去炼丹、种植、守洞府 |
| 全服拍卖行 | 玩家间真实交易，需 NPC 生态支撑 |
| 异步 PvP | 复制玩家数据生成镜像 NPC，丹道大会、斗法榜、宗门比斗 |
| 轮回飞升 | 终局系统。飞升后重修，保留部分底蕴、称号、效率、关系或法宝 |

## 底层系统设计大纲（2026-05-08 最终校准）

> 外围叙事包装（雷劫、洞府、机缘奇遇等）是对底层系统的**事件调用或数据配置**，不单独列系统。以下只列底层系统。

### 系统 1：修炼系统

**定位**：玩家获取修为的核心入口。打坐/闭关/吃丹三条路径共用同一套数值。

**核心数据**：
- `修为值`（Real，累加）
- `修炼速度`（A/B/flat 三池汇总，由 CultBonusManager 当场读表计算）
- `灵气浓度`（洞府属性，日周期波动，作为 B 类来源之一）
- `每日丹药上限`（每种丹药独立上限，防止脚本）

**核心逻辑**：
- 打坐：`每3秒获得修为 = 境界基数×(1+A类)×(1+B类)+Σ固定值`，随时可中断，修炼期间不能战斗
- 闭关：`上线结算 = 离线时长 × 修炼速度 × 离线期灵气均值`，上限 8h（可付费提），不上线不结算
- 吃丹：即时修为直接累加；速率 buff 注册进 CultBonusManager B 类池，限时生效
- 功法差异：修炼功法→B 类加成；丹修炼丹也涨修为（第二个来源，不走速度公式）；剑修斩剑（前置门槛）等

**对外接口**：境界系统查修为判断突破；记录系统读快照；时间系统提供灵气浓度；宗门系统提供修炼加成

**配置项**：基础灵气、灵气峰值时段、峰值倍率（×1.0）、衰减曲线（×0.5→×0.2）、丹药修为表、丹药每日上限

---

### 系统 2：境界系统

**定位**：玩家身份层级。决定属性基数、压制系数、可解锁内容。

**核心数据**：
- `当前境界`（枚举：凡人/练气/筑基/金丹/元婴/化神/渡劫/飞升）
- `当前小阶`（1-10 层，阶段标签：前期1-3/中期4-6/后期7-9/圆满10）
- `突破所需修为`（每境界阈值）
- `突破所需材料`（每境界材料清单，**主药必须来自刷怪**）
- `境界属性基数`（攻击/防御/生命，大境界之间数量级跃迁）
- `压制系数`（高境界对低境界的伤害倍率、减伤倍率）

**核心逻辑**：
- 突破判定：`修为值 ≥ 阈值 AND 材料齐全` → 触发突破事件
- 事件类型：心魔（生成玩家镜像分身）/ 雷劫（多波次天雷）/ 渡劫（组合）
- 成功：境界晋升 + 属性基数重算 + 全屏特效 + 世界播报
- 失败：修为损失 10-30% + 属性临时下降（持续 N 分钟），连续失败 N 次后下次必成
- PvE 压制：`伤害修正 = 1 + (Δ境界) × 0.3`，`受伤修正 = 1 / (1 + (Δ境界) × 0.3)`
- PvP 压制（简版）：系数 ×0.5
- 破压制机制：稀有法宝/极品丹药/特殊功法/机缘传承 → 提升修正系数

**对外接口**：战斗系统查压制系数；记录系统读境界；轮回系统调飞升判定

**配置项**：境界枚举表（阈值/材料清单/属性基数/突破事件类型）、压制系数、失败惩罚参数（修为损失%、临时下降值、保底必成次数）、破压制修正项

### 系统 3：战斗与掉落系统

**定位**：怪物交互 + 资源产出。核心爽感"叮"的发生器。**杀怪不产生经验。**

**核心数据**：
- `怪物属性`（攻击/防御/生命/境界/掉落表 ID）
- `掉落表`（物品 ID × 权重 × 数量范围 × 即食标记）
- `即食类物品`（拾取即转化：修为/临时属性，不进背包）
- `背包类物品`（材料/装备/丹药品阶/功法残卷，进背包）
- `叮的层级`（普通→中等→大→神级，对应不同特效/音效/飘字）

**核心逻辑**：
- 击杀 → 按掉落表权重随机 → 即食类飘字转化（占爆率 90%）→ 背包类进背包（占爆率 10%）
- 即食类：灵气珠（+50 修为飘字）、低阶丹（即吃即涨修为）
- 背包类：突破主药材料、法宝材料、装备、功法残卷
- 高级区 = 掉落表品阶更高 + PvE 压制放水（给偷渡空间）
- 偷渡成功：独立高价值掉落判定

**对外接口**：修炼系统接收即食丹修为；经济系统接收背包材料；境界系统提供压制系数

**配置项**：怪物属性表（按区域/境界）、掉落表（按区域分级）、即食 vs 背包标记、叮的层级特效定义

---

### 系统 4：宗门与长老体系

**定位**：玩家职业/专精身份。决定被动加成、专属技能、经济方向、**修炼方式差异**。

**核心数据**：
- `宗门 ID`（正道/散修/魔道，3 选 1）
- `堂口 ID`（丹堂/器堂/剑堂/阵堂/兽堂，每宗门 5 选 1）
- `被动加成`（属性%/修炼速度%/伤害类型偏向）
- `专属技能`（1 个主动技能）
- `宗门贡献点`
- `修炼方式标记`（影响修炼系统逻辑分支）

**核心逻辑**：
- 首次选择宗门和堂口，可后续花代价更改
- 丹堂：炼丹成功率/品质加成 → 卖丹药赚钱
- 器堂：炼器/制符效率加成 → 卖法宝符箓赚钱
- 剑堂：暴击/攻速/技能伤害加成 → 战斗型 + 打坐+斩剑修炼
- 阵堂：阵法布置/神识探测加成 → 辅助控场型 + 打坐+阵法参悟
- 兽堂：灵兽培养/驾驭加成 → 召唤辅助型 + 打坐+灵兽滋养
- 堂口决定经济方向：产金型（丹/器）vs 战斗型（剑）vs 辅助型（阵/兽），互相依赖（拍卖行交换）

**对外接口**：修炼系统取修炼加成 + 修炼方式差异；经济系统取生产加成；战斗系统取被动加成

**配置项**：宗门枚举、堂口枚举、被动参数表、专属技能定义、修炼方式规则、贡献点兑换表

---

### 系统 5：时间与日夜系统

**定位**：全局时间框架。所有周期行为、事件调度、灵气波动的基础层。

**核心数据**：
- `游戏时间`（与现实映射：如 1:1 或加速，如现实 1h=游戏 4h）
- `当前时段`（清晨/白天/夜晚/深夜）
- `灵气浓度曲线`（逐时段变化）
- `事件调度表`（时间戳 → 事件类型 → 参数）
- `节气节日标记`

**核心逻辑**：
- 循环推进游戏时间 → 更新时段 → 更新灵气浓度
- 事件调度按时间触发：魔界侵蚀预警、特殊刷新窗口、独有怪窗口、节气活动
- 怪物刷新 + 灵植采集查询时段决定可用集合：
  - 清晨（6-9）：灵气浓郁，打坐+50%
  - 白天（9-18）：正常刷怪
  - 夜晚（18-24）：魔物刷新，危险但奖励高
  - 深夜（0-6）：稀有 BOSS 概率高

**对外接口**：修炼系统取灵气浓度；掉落系统取时段影响刷新；机缘系统取触发窗口；屠魔录取侵蚀事件；记录系统读时间戳

**配置项**：周期长度、灵气曲线（峰值/衰减/倍率）、时段定义（含对各系统影响标记）、事件调度表、节气特殊规则

---

### 系统 6：经济生产系统

**定位**：材料→成品 + 物品流通。含炼丹、炼器、拍卖行单机层。

**核心数据**：
- `丹药配方`（材料清单 → 产出 ID × 品质权重）
- `法宝配方`（材料清单 → 产出 ID × 品质权重）
- `自动炼丹队列`（材料进队列 → 计时器 → 普通丹药产出）
- `拍卖行单机存货`（物品 ID × 价格 × 存量 × 刷新周期）

**核心逻辑**：
- 自动炼丹（前期）：材料进队列 → 按配方计时产出普通丹，无品质波动
- 手动炼丹（后期）：玩家操作（配比/火候）→ 品质判定（普通/极品/废丹/异变丹）→ 极品效果 5-10 倍，可高价拍卖
- 自动炼器：同上模式
- 拍卖行单机层：NPC 挂单，系统定价 × 定期刷新，保证单人经济流畅

**对外接口**：宗门系统提供炼丹/炼器加成系数；掉落系统提供材料输入；修炼系统消耗丹药

**配置项**：配方表（材料/产出/时长）、品质权重表、拍卖行定价规则/刷新周期

---

### 系统 7：机缘系统

**定位**：稀有高价值事件集合。独有怪物、蓝门副本、奇遇事件。

**核心数据**：
- `独有怪物刷新标记`（每类型 boolean，已刷新 = 永不刷新）
- `机缘触发表`（条件 → 事件 ID → 奖励）
- `蓝门触发条件`（击杀计数阈值 / 特定地图 / 特定时段）

**核心逻辑**：
- 独有怪物：每类型全局仅 1 次刷新 → 被杀后永久标记 → 掉落唯一物品 + 世界播报
- 蓝门：`击杀计数达阈值 ∨ 特定地图小概率 ∨ 特定时段` → 传送门 → 独立副本（剑冢/丹谷/魔窟）→ 一次性，通关消失
- 奖励走独立高价值表，不走常规掉落表

**对外接口**：时间系统提供触发窗口；掉落系统提供击杀计数；记录系统记录机缘事件

**配置项**：独有怪物表（概率/掉落/刷新条件）、蓝门触发条件/主题/奖励表、奇遇事件表

### 系统 8：屠魔录系统

**定位**：大秘境分层挑战 + 全服排名。周期性 PvE 事件，给玩家外部目标和史诗叙事。

**核心数据**：
- `魔界侵蚀状态`（进行中/剩余时间）
- `玩家当前层数`
- `秘境怪物属性`（层数 × 系数递增）
- `排行榜`（最高层数 ↔ 用时排序）

**核心逻辑**：
- 时间系统按预设周期触发侵蚀 → 持续 N 小时 → 开放秘境
- 秘境分层（1-N 层）→ 每层怪物递增 → 通关 = 清场 → 解锁下一层
- 排行榜全服可见 → 赛季结算奖励

**对外接口**：时间系统触发侵蚀事件；战斗系统加载秘境怪物；记录系统写入排行榜

**配置项**：侵蚀周期/持续时长、层数上限、每层怪物系数、排名规则、赛季奖励表

### 系统 9：记录系统

**定位**：自动生成玩家个人史和战斗报告。**传播命脉**——玩家自发分享。

**核心数据**：
- `玩家事件日志`（事件类型/时间戳/数值快照/NPC 关联）
- `修仙录模板`（突破/机缘/击杀/交易/关系变化记录）
- `道争录战报模板`："[A]以[境界A]之姿，[N]招击败[B]；关键一击：[技能]造成[数值]伤害"

**核心逻辑**：
- 自动记录：突破/击杀独有怪/机缘触发/NPC 关系变化/拍卖战绩
- 飞升时遍历日志 → 生成"此生大事记"长图（可分享至社区）
- 道争录：异步 PvP 数据 → 战报模板填空

**对外接口**：所有系统写入事件日志；社区/分享模块读取修仙录生成传播素材

**配置项**：事件类型枚举、修仙录模板定义、道争录战报模板

### 系统 10：轮回系统

**定位**：终局目标，飞升后可选轮回。P3 实现但 **P0 预留**。

**核心数据**：
- `轮回层数`（初始 0）
- `轮回永久加成系数`（层数 × 固定比例，影响属性基数和修炼速度）
- `轮回称号`（三世修者/九转金仙等）

**核心逻辑**：
- 飞升后可选轮回 → 重置常规进度 → 保留轮回层数
- 每轮回永久属性加成 + 修炼速度加成
- 高层轮回解锁新内容（3 世解锁魔道、10 世解锁开宗立派）

**P0 预留**：角色数据结构加"轮回层数"字段（初始 0）；所有属性公式预留"轮回加成系数"接口（初始 1.0）；修仙录日志预留 `is_retained_across_reincarnation` 标记；存档格式预留 `version` + `extension_data` 扩展区

**对外接口**：境界系统调飞升判定；存档系统读写轮回数据；所有属性公式接入加成系数

**配置项**：轮回层数上限、每层固定比例加成长、解锁内容阈值、轮回专属称号/特效

### 系统 11：NPC 交互系统

**定位**：世界生活感。友好度/家奴/江湖传闻。P3 远期。

**核心数据**：
- `NPC 基本信息`（ID/名称/境界/宗门/位置）
- `玩家-NPC 友好度`（-100~100）
- `家奴列表`（NPC ID × 功能角色 × 派遣状态）

**核心逻辑**：
- 友好度：通过交互/送礼/共同御敌等行为增减
- 家奴收服：可收服魔性 NPC → 绑定到玩家 → 不可交易 → 派遣（炼丹/种植灵植/守护洞府/战斗助阵）

**对外接口**：记录系统读取 NPC 关系变化写入事件日志；洞府系统接入家奴加成

**配置项**：NPC 属性表、友好度增减规则、家奴功能类型、传闻触发表

**对外接口**：记录系统写入战报；宗门系统调排行榜

**配置项**：镜像复制规则、擂台类型、赛季结算
