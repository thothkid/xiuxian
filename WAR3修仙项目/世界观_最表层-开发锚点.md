## 零、卷首总述

### 【一句话】

这是一个由天界大阵格式化灵气、魔界裂隙泄漏怨化魔气、神魔大战废墟散布其间的修真世界。玩家是边荒小宗门青冥宗的一名外门弟子——刚刚引气入体的那一刻。

### 【玩家第一刻】

你睁开眼，经脉里第一次涌动着灵气的暖流。眼前的世界不一样了——你能"看见"空气中的灵气在流动。师兄递给你一本入门功法，管事登记了你的灵根，给了你一处简陋洞府的钥匙。白天出门斩妖，傍晚回洞府炼丹，夜里山外魔气起伏，偶尔有蓝色秘境裂隙在远处闪烁。第二天上线，闭关的修为涨了一截——但真正改变命运的，还是昨晚冒险带回来的主药和法宝。

百年后你可能青冥宗的金丹长老，千年后你或许是化神老怪，万年后九重雷劫之下飞升而去——或者轮回重来。

### 【不做什么】

❌ 不做天命救世主 ❌ 不做强制主线一本道 ❌ 不做纯挂机成仙 ❌ 不做一局速通 ❌ 不做剧情压过刷怪 ❌ 不做宗门绑死成长 ❌ 不做必须组队推进的核心进度

## 一、世界结构

### 【世界叙述】

宇宙有三个位面。**玄黄界（人间界）**——玩家所在的物质世界。天界大阵覆盖其上，灵气被格式化后流转于灵脉网络。魔气从裂隙渗入，神魔大战的废墟散布其间。**天界**——最早一批拒绝大轮回解散的秩序生命建立的存在机器。不是天堂，而是一个为了延续自身而不断运转的秩序系统。飞升通道是它回收人间天才的管道。**魔界**——被天界封印了百万年的混沌位面。封印制造了魔气，魔气制造了魔族。万年前裂隙撕开，战争至今未停。

### 【开发锚点】

| 位面 | 系统对应 |
|---|---|
| 人间界 | 玩家全部日常活动舞台 |
| 天界（大阵+飞升+轮回） | N突破/A基础架构/M境界/AR终局轮回 |
| 魔界侵蚀 | E怪物词缀/Z魔化层级/AH屠魔录 |

## 二、时代背景

### 【世界叙述】

万年前神魔大战撕裂了天地。飞升通道崩塌，天界大阵出现大量盲区。此后万年，原初秩序和原初混沌开始在某些地方渗回人间界。近千年劫气渐散，被封禁的上古洞府开始松动，蓝光传送门偶尔在山崖上裂开。修士在古战场挖出不属于任何炼器流派的碎片——没人知道它们来自哪一方，但它们携带的能量至今没有完全消散。

## 三、四种气

### 【世界叙述】

修士日常感知的"灵气"其实是四种气的其中之一。真正流过这个世界的气有四类：

**原初秩序**——大轮回吸气的自然产物。庞大、无方向、携带宇宙源头完整记忆。未经天界大阵处理。修士经脉和丹田是为格式化后的灵气设计的，强行吸收原初秩序会经脉紊乱。但一旦建立起新的承载结构，可开辟天界功法之外的全新路径。仅存于大阵盲区——古战场深处、仙路断裂处、封禁洞府。

**原初混沌**——大轮回呼气的自然产物。中性、流动、负责解构与回收。它不是邪暗属性，而是自然回收力。与修士秩序结构冲突——它流过秩序时会叫秩序回归自然循环。仅存于魔界极深处和天界控制盲区。

**灵气**——原初秩序被天界大阵格式化后的产物。有方向、有节奏、易吸收、稳定。但被屏蔽了源头信息——修士吸收时不接收大轮回信号。修士日常修炼的版本，上限被天界预设路径锁定。天界大阵覆盖区域的全部灵脉都流淌着这种灵气。

**魔气**——原初混沌被封印百万年后，在压力、隔绝与怨恨中变质的污染形态。高能量但紊乱，携带被封者的怨恨信号。正常修士直接吸收会严重损毁经脉。极少数魔修以特殊功法在体内建立隔离结构微量利用。从魔界裂隙泄漏，污染魔化区域和古战场。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 四种气的区分 | 所有物质系统的底层逻辑——不同区域产出不同气态对应不同资源 |
| 灵气=格式化后的原初秩序 | L修炼——修炼路径被锁定在标准体系内 |
| 魔气=囚禁怨化后的原初混沌 | Z魔化区域特殊规则、魔抗属性需求 |
| 大阵盲区中的原初秩序 | AG机缘秘境——高阶玩家的"不自然"感知 |

## 四、日月、阴阳与浊气

### 【世界叙述】

**日灵气**——白昼形态的灵气，偏向生长、外放、升腾。适合灵植生长、血肉活化、火性功法。**月灵气**——夜晚形态的灵气，偏向凝聚、内敛、沉降。适合妖类吐纳、草木凝药、神识沉淀。

**浊气**不是魔气。浊气是白日过盛、地热上升、尘土、杂乱灵流混合成的燥烈之气。它带有混乱倾向但还没变成魔气。魔族和魔化生物更喜欢浊气环境——浊气像干柴，魔气像火星。没有裂隙时只是燥乱，有裂隙时加速魔化。

一天节律：清晨灵气清明（打坐）、正午日灵气盛但浊气升（办事）、傍晚灵气回落（整理）、夜间月灵气主导（妖兽活跃）、深夜魔气渗出（风险收益双高）。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 清晨灵气最纯 | Y时间系统——清晨打坐加成 |
| 浊气≠魔气，但易被引动 | Z魔化区域——浊气浓度影响魔化刷新 |
| 夜间月灵气→妖兽活跃 | D怪物刷新——夜间特殊刷新 |
| 深夜魔气渗出→稀有怪 | AG蓝门秘境触发、E词缀怪概率 |

## 五、五行与物质结构

### 【世界叙述】

五行不是五种化学元素，是灵气被梳理后呈现出的五种运动形态——金（锐利收束）、木（生长舒展）、水（流动渗透）、火（爆发交换）、土（承载凝聚）。生克不是绝对胜负——境界、灵气密度、功法理解都能压过五行关系。

灵根决定修士更容易读懂哪一种五行灵气。单灵根读得快但单一，多灵根适应性广但慢。异灵根（风雷冰）是五行在特定节律下的复合形态。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 五行=灵气运动形态 | 所有材料/怪物/地图挂五行标签 |
| 生克关系 | C战斗伤害修正、S炼丹成功率修正 |
| 灵根=灵魂秩序框架 | O灵根系统——修炼效率的区域差异 |

## 六、修士身体观

### 【世界叙述】

**灵根**——灵魂中自带的天然秩序框架。没有灵根的凡人无法识别灵气结构——像风吹过石头，不会被留下。**经脉**——灵气通道，小境界突破时扩宽加固。**丹田**——灵气压缩仓，把稀薄天地灵气压缩成灵液/金丹。**神识**——读取外界灵气扰动的能力，小地图范围+隐藏探测+法宝操控数。**法力**——可调用的压缩灵气。**真元**——金丹以上与道基绑定的本源灵气，日常法术不消耗，只有本命法宝祭炼、渡劫、心魔才动真元。**寿元**——体内秩序结构的自然维持周期，大境界突破整体加固。化神出手损寿=体内秩序与外界共振消耗。**道心**——对自身秩序路径的稳定认同，心魔劫攻击的是道心中的裂缝。

### 【开发锚点】

| 灵根=灵气吸收通行证 | O灵根系统 |

| 冠/巾=境界身份外显 | H装备位——低构筑·高填充位 |
| 法袍/道袍=身披道法 | H装备位——生存+外观，题材包装位 |
| 武器=手持战力 | H装备位——攻击方式选择，高构筑位 |

## 八、妖兽、精怪与气灵

### 【世界叙述】

妖兽是动物长期受灵气改造的结果。一个普通山狐世代生活在灵脉附近，血肉骨骼被灵气重排——更壮更快更硬。妖丹不是它刻意修炼的，是灵气被体温、心跳、猎杀这些生命活动自然加工后沉淀在体内的结晶。修士炼化妖丹，本质上是借用了妖兽数十年数百年对灵气的理解成果。

**妖修分三类**：妖兽（肉身路线）、精怪（草木玉石灵气结构复杂到涌现意识——和器灵诞生逻辑一致）、气灵（高浓度灵气临时聚成的短暂意识）。

妖兽阶位1-9阶对应练气到渡劫。四阶以上有兽魂——肉体死亡后短暂保留的神识结构。五阶可通人言，六阶可化形。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 妖兽1-9阶+产出全表 | D怪物刷新/F掉落表 |
| 妖丹=生命解读成果 | S炼丹主材/N突破主药 |
| 精怪=草木成灵 | AA NPC（稀有交互）/AK灵兽 |

## 九、魔化生物与魔核

### 【世界叙述】

魔化不是"妖兽变坏"。魔化是魔气侵入已有生命结构后对其秩序进行扭曲。**魔化生物**——原本人间界生物被魔气污染后异变，不再按本能行动。**魔界原生生物**——从魔界裂隙中爬入的异位面生物，结构本就不适合人间秩序。

**魔核不是妖丹染黑。** 魔核是魔气在生物体内重构出的混乱核心——能量高但信息难以读懂。正常修士不能直接炼化，但炼器师、魔修、屠魔录奖励体系可利用其高能量。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 魔化生物 vs 魔界原生生物 | D怪物分类——词缀怪 vs BOSS |
| 魔化生物不恐惧不休息 | D怪物AI |
| 魔核≠妖丹 | F掉落——两套掉落体系 |
| 魔核高能不可直接炼化 | T炼器特殊材料/AH屠魔录奖励 |

## 十、灵植体系

### 【世界叙述】

灵植不是"草药加灵气"。灵植是一套活的灵气循环系统——根吸收地气水气→茎输送→叶吸收日灵气→花果在月灵气中凝聚药力。同样的遗传密码，灵气充足时表现为灵植，灵气不足时表现为普通植物。

采集就是打破这个循环——卯时采叶（药性发散），子时采根（药性沉积）。断根须不是少了一点重量，是断了一条灵气通路，整个循环失衡。

**四性**——寒热温凉是灵植体内灵气的主流流动方向。**五味**——酸苦甘辛咸是灵植灵气与人体五行回路的天然亲和，决定药力先入哪条经脉。**药效八类**——补气/凝神/淬体/洗髓/破境主药/解毒清心/火候药引/魔化灵植。**年份**——十年凡品/百年灵品/千年仙品/万年神品。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 四性五味归经 | S丹药配方底层逻辑 |
| 八类药效 | S丹药分类——补气→修为丹/破境→突破丹 |

## 十一、炼丹

### 【世界叙述】

炼丹不是化学反应。炼丹是把不同材料中已整理好的灵气结构拆开、调和、重组。火候是用火性灵气的高频交换打破材料外层灵气膜，让君臣佐使之间重新配对。

**丹毒**——炼丹时四性五味的灵气结构被火候强行撮合，总有一部分被打散后找不到配对，变成无方向的小碎片附着在经脉壁上。打散速度和重新配对速度不可能完全同步——这是热力学问题，不是技术问题。丹毒缓慢排出，但吃丹速度远超排出速度——这就是每日上限的物质法则解释。

**丹药九品**：一品到九品效果和丹毒递进，倍率示意以数值表为准。奇数品是翻倍点——五品进入坊市、七品拍卖行级别、九品传说级全服播报。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 君臣佐使+四性五味 | S丹药配方系统 |
| 火候与丹火温度 | S手动炼丹品质判定 |
| 丹毒与每日上限 | L修炼——每日丹药上限 |
| 丹药九品 | S品质分级/掉落/商店/AD拍卖 |

## 十二、炼器、法宝与道器

### 【世界叙述】

**凡器→法器→灵器→法宝→本命法宝→道器**。炼器和炼丹同源——用火候打破材料原有结构再让灵气重排。区别在于炼丹重组药力进身体，炼器重组物质成外物。

灵器压缩到极致时材料表面会出现纹路——不是工匠刻的，是材料内部晶体排列的自然外显。这些"器纹"或"灵纹"就是装备词条的世界观来源——每件材料的淬炼环境微差造就不同纹路。

法宝不是造出来的，是养出来的。胚子打造后，修士用自己的灵力、精血祭炼，胚子内部的灵气循环逐渐与主人经脉频率同步——这就是认主的物理本质。本命法宝随主人境界跃迁被带动升级，但维持共振消耗真元——大多数修士撑不住两个以上。

**道器**不按普通炼器逻辑复制。它是上古秩序结构碎片——上古法宝、阵法核心、古修道痕——在特殊条件下被保留下来，成为独一无二的生涯级奇物。不可交易不可复制。

**器灵**——当法宝内部灵气循环复杂到临界点，意识会涌现。和精怪诞生逻辑完全一致：秩序结构复杂到一定程度便可能"知道自己在"。最古老的器灵是上古修士的本命法宝——主人陨落后法宝未碎，继续在炉子旁吸收残余灵气，千年后醒了，自己会选下一任主人。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|---|
| 凡器→法器→灵器→法宝→本命法宝→道器 | H装备分层/Q法宝/R道器 |
| 器纹/灵纹=装备词条 | J词条系统/I小极品 |
| 法宝认主=频率同步 | Q法宝祭炼/认主 |
| 本命法宝=真元共振+跃迁推升 | H装备跨境界成长 |
| 道器=上古秩序残留 | R道器系统——不可交易/轮回重置 |
| 器灵=秩序涌现意识 | Q法宝高阶养成/AK灵兽同类逻辑 |
| 万物皆可炼化为法宝 | 多轴组合（器物外形+功能类+释放形态+五行+标志性规则），详见《法宝行为分类与隐藏分类体系》 |

## 十三、符箓与阵法

### 【世界叙述】

符箓和阵法的底层是同一件事——把灵气结构固定在载体上。符箓固定在纸/兽皮/玉片上，阵法固定在空间里。

**符箓**——法术结构的临时缓存。正常施法需要临时构建灵气结构，符箓把这一步提前做好了，使用时只需输入少量法力触发。符纸用灵木压制——灵木曾经有过灵气循环轨迹，类似预格式化的光盘。符墨用妖血——妖血携带妖兽对灵气的解读成果，比人血更适合穿透符纸纤维。

**阵法**——空间化的符箓。阵旗是锚点，阵盘是中枢，灵石是持续供能。聚灵阵（漏斗形汇聚灵气）、护府阵（边界灵气膜）、困阵（提高灵气密度使目标体内灵气与外界互斥）、陷阱阵（预存灵气结构）、迷阵（干扰神识）、古传送阵（灵气波动解码空间锁）、禁制阵法（灵气加锁）。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 符箓=提前缓存法术 | T符箓制作——偷渡/战斗消耗品 |
| 符纸/符墨/符笔 | U通用合成——材料→符箓 |
| 阵法七类 | V阵法系统——阵旗阵盘+灵石 |
| 弱者靠阵胜强者 | V战斗阵法——Buff机制实现 |
| 聚灵阵/护府阵 | V洞府阵法/W洞府入口 |

## 十四、灵石与矿产

### 【世界叙述】

灵石是灵气在地层中长期凝结后的晶体——整块石头被灵气格式化成特殊晶体结构，品阶越高压缩程度越高。下品（日常消耗）、中品（辅助修炼+阵法）、上品（高级阵法+高阶丹炉）、极品（接近固态灵气核心，拍卖行硬通货）。

矿产的本质是物质晶体被灵气长期重排——玄铁（金气重排铁矿）、精金（极限致密）、陨铁（天外材料+灵气重构）、灵木（内部形成稳定灵气回路）、玉髓（灵石变体，适合承载灵气信息）、五行精华（单一五行灵气极致压缩）。

### 【开发锚点】

| 设定项 | 游戏机制 |
|---|---|
| 下/中/上/极品灵石 | AD货币体系——灵石分级 |
| 灵石多用途 | L修炼/S炼丹/V阵法/N突破消耗 |
| 矿产=灵气重排物质晶体 | T炼器材料/H装备制作 |
| 五行精华 | N突破/T炼器巅峰材料 |

## 十五、万宝坊市与拍卖行

### 【世界叙述】

万宝坊市是三不管地带的一小片平地——青冥宗和古战场之间，哪边都管不着。散修自发摆摊形成的交易点。规则简单：摊主摆货交灵石、买家自己看货、出门不管。

捡漏是核心——摊主走眼低价挂了宝贝，你识货就赚翻。看走眼买了被魔气轻微污染的材料，回去炸炉——自己学艺不精，没处退。匿名挂单是因为货来路不明——偷的、捡的、死人的遗物。

拍卖行是另一套玩法——大宗稀有品（道器碎片、上古功法残卷、七品以上丹药、本命法宝胚子）走竞价。手续费是经济回收口。

## 十六、玩家身份与起点

### 【世界叙述】

青冥宗——东玄洲边荒的一个小宗门。山门有石阶十二阶，广场立着先代祖师石像。外门弟子居所在西山脚下——石屋漏风但能住。去过中州大型坊市的人再看青冥宗，觉得跟人家柴房差不多。但这不重要——你是青冥宗的弟子，丹田里涌动着练气一层的灵气，这是你已经跨过去的门槛。

你不需要"加入"宗门，你已经在里面了。筑基→内门自动晋升。长老不独占弟子——不存在"拜入某长老门下"的排他绑定。想向剑堂长老学剑道、向丹堂长老学炼丹、向阵堂长老修神识，都可以——关系深浅看你的境界、友好度、宗门声望。达到门槛，服务开放。这是加法：和五位长老都有深浅不同的关系，你的修炼路线是五条线的交织。

## 二十四、术语与命名规范

### 【核心术语】

| 术语 | 精确定义 |
|---|---|
| 灵力 | 修士体内可调用的灵气总量（=蓝量） |
| 修为 | 长期积累的修炼进度，不可消耗（=经验条） |
| 法力 | 释放法术时被消耗的压缩灵气（=技能蓝耗） |
| 神识 | 读取外界灵气扰动的能力（=小地图+隐藏探测+法宝操控数） |
| 真元 | 金丹以上与道基绑定的本源灵气 |
| 寿元 | 体内秩序结构自然维持周期（=氛围数值） |
| 道心 | 对自身秩序路径的稳定认同（=心魔抗性） |

### 【品质与色系】

品质四档：凡品(白)/灵品(绿)/仙品(蓝)/神品(金)。稀有度独立于品质：普通/稀有/史诗/传说。境界色系语义递进：低境界质朴→中境界富贵→高境界沉稳→终局超脱。具体色值交视觉规范。

### 【命名规则】

丹药按"属性+药效+品阶"如"寒·筑基丹·三品"。功法按"意境+类型"如"青冥剑诀"。本命法宝由玩家自命名。
