## 一、V0 口径锁定

### 1.1 V0 目标（一句话）

**服务器存档下的灰盒最小闭环**：进房 → 建角 → 灵根可见不生效 → 进灰盒地图 → 打怪 → 掉落 → 拾取/背包 → 保存 → 退出 → 重进数据一致。

### 1.2 V0 完整闭环

```
玩家进房(KK入口) → 角色创建(命名+灵根5选,不生效) → 进入灰盒(安全区出生)
   → 打怪(普攻必中) → 怪物死亡掉落 → 地面掉落物(60s可拾)
   → 拾取/背包(右键使用/穿戴) → 服务器存档(本地+云双写) → 退出+重进数据一致
```

### 1.3 V0 做什么（正面清单）

| # | V0 做 | 范围 |
|---|-------|------|
| 1 | 房间系统 | 创建/加入/退出、房主踢人/开始、断线 60s 保留 |
| 2 | 角色创建 | 角色名校验 → 灵根 5 选 1（仅展示+存档）→ 写服务器初始数据 |
| 3 | 基础属性 | 生命/灵力/攻击/防御 4 项生效。**严禁力量/敏捷/智力** |
| 4 | 战斗系统 | 普攻必中，伤害 = max(攻击-防御,1)，白色飘字，无暴击/闪避/抗性 |
| 5 | 怪物系统 | 3 种修仙测试怪。六态 AI：待机→索敌→追击→攻击→脱战→归位→死亡。按刷新表定时重刷 |
| 6 | 掉落系统 | 怪物死亡→读掉落表→32 槽权重随机→地面生成→60s 消失。归属判定 |
| 7 | 物品与背包 | 10 种修仙物品。拾取入格/堆叠/右键使用/穿戴。6 格（可配 12）。满包提示 |
| 8 | 服务器存档 | 本地+云双写，CRC 校验，version 字段。退出重进完整还原 |
| 9 | 配置表加载 | JSON，统一接口，字段缺失报错（不静默）。8 张 V0 配置表 |
| 10 | 基础 UI | 创角界面 + HUD（血/灵/修为占位条）+ 背包界面 + 系统提示 + 伤害飘字。WC3 原生风格 |

### 1.4 V0 边界原则（为什么不做的判断标准）

| 原则 | 说明 |
|------|------|
| 闭环优先 | 不阻塞"进房→建角→打怪→掉落→拾取→存档→重进"链路的，一律不做 |
| 灰盒可测 | V0 唯一目的是跑通技术链路。需完整内容才有效果的系统（境界突破/炼丹/任务）一律后置 |
| 属性不可逆 | 属性体系一旦写错（如用力量/敏捷/智力），后续全部配置表返工。V0 只做 4 核心属性 |
| 存档一次定型 | 存档字段结构一旦发布不能轻易动。V0 写死字段（灵根ID/命格/道器/轮回占位）已考虑 V2-V6 扩展 |
| 老表结构参考，内容置换 | 老项目数据组织方式（列定义/权重池/ConfigRect）已验证可用，但旧属性名、旧传奇枚举一律不进 V0 |

## 二、Nope 清单（十大类）

| 类别 | 不做内容 | 归入版本 |
|------|---------|---------|
| 修炼与境界 | 打坐/闭关/境界突破/雷劫/心魔/修为增减 | V2 |
| 灵根 | 灵根数值效果/纯度/五行亲和生效/变异灵根(暗/光) | V2 |
| 命格/道器/轮回 | 命格系统/道器效果/轮回逻辑 | V2/V5/V6 |
| 炼丹 | 炼丹系统/丹方/品质/丹毒 | V4 |
| 装备 | 完整装备体系/品质分级/强化/套装/词条 | V3/V4 |
| 技能 | 主动技能/CD/弹道/暴击/闪避/抗性/Buff | V1/V3 |
| 怪物 | 精英怪/BOSS/词缀/复杂AI(巡逻/呼救/逃跑/技能) | V3 |
| 背包 | 自动拾取/整理/拆分/仓库 | V1/V3/V4 |
| 社交 | 宗门/洞府/拍卖/交易/排行榜/PvP | V5/V7 |
| 剧情 | 任务/引导/NPC/机缘/秘境 | V1/V5 |
| 美术 | 修仙风格UI/高品质图标/技能图标/时装/坐骑 | V1+ |

> **例外（背包·自动拾取）**：灵石 V0 起自动收取——不占格、无决策空间，视为「地面货币」而非「地面物品」，故不受本行「自动拾取→V1」约束。其余物品 V0 仍手点拾取。见 P08 §六 / §9.3。

### V0 最多占位（界面可见，不生效）

- 修为条（HUD 灰色，不增长，标"V2开放"）
- 境界（固定"练气一层"）
- 灵根 ID（存档不生效）
- 命格/道器/轮回字段（占位 int）

### V0 属性体系（4 项生效）

| 属性 | 字段 | 初始值(练气一层) | 显示位置 |
|------|------|-----------------|---------|
| 生命值 | hp | 100 | HUD血条 + 角色面板 |
| 灵力值 | mp | 50 | HUD蓝条 + 角色面板 |
| 攻击 | atk | 10 | 角色面板 + 装备Tooltip |
| 防御 | def | 5 | 角色面板 + 装备Tooltip |

伤害公式：`damage = max(攻击 - 防御, 1)`，必中，无暴击/闪避/抗性。

> 标注规则：**保留结构**=列定义/权重池/坐标方式可复用，内容全换修仙；**后置**=结构留待后续版本；**丢弃**=传奇特有，V0 不碰。

| 老表 | V0 处理 | 对应 V0 配置表 | 说明 |

| 属性配置表V1.01 | 保留结构 | attr_config | config_id+attr_key+data_type 结构复用，属性枚举全换 |
| J角色属性1.1(Lv1行) | 保留结构 | init_char_config | 初始值一行复用，属性换单值修仙属性 |
| G怪物表1.43(590怪70列) | 保留结构 | monster_config | →3 怪 10+ 列，攻击上下限→单值 |
