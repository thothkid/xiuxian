## 二、属性类型说明

| 类型 | 说明 |
|------|------|
| BASE | 基础属性，直接参与伤害公式或系统计算 |
| MODIFIER | 修正属性，由属性管理器预先计算到对应的 BASE 属性上，不直接出现在公式中 |

MODIFIER 生效规则：`最终值 = 父级BASE值 × (1 + 增幅)` 或 `最终值 = 父级BASE值 + 增加值`（取决于 InfluenceOP 为 MUL 或 ADD）。攻法下限/上限特殊：同时 ADD 到攻击和法力两个父级。

---

## 三、数据格式约定

- 万分比属性：存储为整数，计算时除以 10000。如 10000 = 100%，500 = 5%
- 整数属性：直接使用存储值
- 上下限属性：下限 ≤ 上限。天运触发时取上限，未触发时在下限～上限间随机
