# 新手任务系统设计文档

## 1. 文档信息

| 项目 | 内容 |
|---|---|
| 文档类型 | 新手任务系统设计文档 |
| 内容范围 | 主线任务 1.1-1.8，从加入宗门到首次筑基 |
| 内容底稿 | 《新手任务流程222.md》及原始流程图 |
| 核心目的 | 完成世界观导入、基础操作教学、核心养成教学及首次境界突破 |
| 目标玩家 | 首次进入游戏、不了解基础操作和养成系统的新玩家 |
| 当前状态 | 流程结构化版本；数值、资源名称及部分系统规则待确认 |

## 2. 设计目标

1. 在一条连续的宗门成长线中，逐步教授移动、战斗、拾取、背包、炼丹、功法、修炼、任务和突破系统。
2. 每个任务原则上只引入一个主要新系统，避免同时塞入过多信息。
3. 使用 NPC 引导和角色动机包装功能教学，减少纯 UI 说明带来的割裂感。
4. 每完成一个任务，都给予明确反馈：任务完成提示、奖励、系统解锁或角色成长。
5. 最终通过首次筑基形成阶段性高潮，并明确后续长期成长方向。

## 4. 标准任务数据结构

后续任务配置建议统一包含以下字段：

| 字段 | 用途 |
|---|---|
| 任务 ID / 名称 | 唯一标识与玩家侧显示名称 |
| 任务类型 | 主线、支线、世界、宗门、悬赏等 |
| 前置条件 | 前置任务、等级、境界、道具或区域条件 |
| 触发方式 | 自动触发、NPC 对话、区域触发、系统事件 |
| 开始/结束 NPC | 任务接取与交付对象 |
| 任务地点 | 涉及场景、坐标、传送点和副本 |
| 玩家目标 | 玩家可见的目标文案 |
| 子目标 | 可追踪、可计数的分步目标 |
| 玩家操作 | 玩家必须完成的交互和输入 |
| 任务对话 | 按角色、节点和条件配置的对白 |
| 设计意图 | 教学目的、情绪目的和节奏目的 |
| 系统支持 | 任务、战斗、背包、教学、存档等程序能力 |
| 表现资源 | NPC、场景、动画、特效、音效、UI、文本 |
| 异常处理 | 断线、退出、物品丢失、失败和重复触发处理 |
| 完成条件 | 服务端/客户端用于判定任务完成的条件 |
| 奖励与解锁 | 道具、修为、功能、称号、后续任务 |
| 验收标准 | 测试时可明确判断通过或失败的标准 |

## 5. 共用系统支持

### 5.1 任务系统

- 主线任务链及前置条件管理。
- 自动触发、NPC 触发、区域触发和系统事件触发。
- 单步目标、复合目标、计数目标、道具目标和等级目标。
- 任务追踪、导航、场景切换后的状态恢复。
- NPC 头顶可接取、进行中、可交付标识。
- 任务中断、掉线重连、放弃和重新接取处理。
- 任务奖励发放及背包满时的补发机制。

### 5.2 新手教学系统

- 强引导遮罩、按钮高亮、箭头、操作提示和完成检测。
- 教学步骤不可跳过/可跳过规则。
- 教学触发防重复、失败重试和跨端输入适配。
- 每次只突出当前需要操作的目标，完成后自动进入下一步。

### 5.3 对话与演出

- 多角色对话、角色名、头像、立绘和对白顺序。
- 对话期间锁定/解锁移动与镜头控制。
- 条件对白、任务前后不同对白及重复交互对白。
- 必要的转身、指向、打量、搭脉等轻量动作。

### 5.4 场景与导航

- 山门、大殿广场、灵植园、炼丹房、功法阁、勤务堂、渡劫台等导航点。
- 世界区域与副本之间的传送及返回点记录。
- 路径指引、目标距离、区域触发器和防卡死传送。

### 5.5 成长与存档

- 修为、境界、功法熟练度、丹毒、任务道具等数据持久化。
- 新手阶段关键节点实时存档。
- 任务奖励、掉落与解锁功能的幂等处理，防止重复领取。

## 6.1 主线任务 1.1：加入宗门

### 基础信息

| 项目 | 内容 |
|---|---|
| 任务类型 | 主线/新手教学 |
| 前置条件 | 创建角色并完成背景滚屏 |
| 触发方式 | 出生自动触发 |
| 开始 NPC | 莫黎长老 |
| 结束目标 | 找到接引弟子陆青 |
| 地点 | 山门牌坊、大殿广场 |
| 核心教学 | 移动、镜头、NPC 交互、任务追踪 |

### 玩家可见目标

1. 与莫黎长老交谈。
2. 学会移动和镜头操作。
3. 前往大殿广场。
4. 找到接引弟子陆青。

### 流程与玩家操作

1. 背景简述滚屏结束，玩家出生在山门牌坊下。
2. 弹出移动教学：使用 WASD 或鼠标点击移动，使用滚轮缩放镜头。
3. 玩家靠近并点击莫黎长老，触发“拜入宗门”对话。
4. 对话结束，显示陆青的任务追踪与导航。
5. 玩家抵达大殿广场并找到陆青，进入任务 1.2。

### 任务对话

**莫黎长老：**

> 老夫知晓你痛失故土、亲人尽亡，心中悲苦无处安放。可一味消沉，只会让屠戮村落的魔修逍遥法外。从今往后，天元宗便是你的容身之处，好好修炼，待他日修为有成，既可探寻灵宝下落，亦可手刃仇敌，告慰全村亡魂。

**玩家：**

> 晚辈谨记长老教诲。过往伤痛藏于心间，不再沉沦悲戚。从今往后，一心问道，刻苦修行，以手中修为护正道。

**莫黎长老：**

> 见你能振作起来，老夫心中甚是宽慰。你方才拜入我天元宗，如今尚且只是外门弟子，外门一众新人皆由陆青负责接引。你现在便前去寻他报到吧，安排稳妥后再来功法阁寻我。

### 设计意图

- 用灭村、复仇和寻宝建立玩家的长期动机。
- 让移动教学自然发生在“前往报到”的行动中。
- 建立莫黎长老作为主线导师、陆青作为新手接引人的角色关系。

### 所需支持与资源

- 移动及镜头教学 UI。
- NPC 点击交互、头顶任务标识和寻路指引。
- 背景滚屏、山门出生点和大殿广场导航点。
- 莫黎长老、陆青模型/立绘及基础待机动作。

### 完成与验收

- 玩家完成莫黎长老对话并抵达陆青附近后完成任务。
- 教学必须检测到至少一次有效移动输入。
- 退出重进后不得重复播放背景滚屏或重复发放奖励。

## 6.2 主线任务 1.2：试炼幻境——首次击杀

### 基础信息

| 项目 | 内容 |
|---|---|
| 前置条件 | 完成任务 1.1 |
| 触发方式 | 与陆青对话 |
| 开始/结束 NPC | 陆青 |
| 地点 | 山门广场、试炼幻境 |
| 核心教学 | 背包、战斗、拾取、装备、药品 |

### 玩家可见目标

1. 与陆青交谈并领取储物袋。
2. 进入试炼幻境。
3. 击杀 4 只“XXXX”。
4. 拾取掉落物。
5. 打开背包，穿戴装备或服用药品。
6. 返回陆青处交付任务。

### 流程与玩家操作

1. 陆青赠送储物袋，解锁背包格子并引导打开背包。
2. 对话后将玩家与陆青传送至试炼幻境。
3. 教学左键选中目标、右键攻击、QWER 使用技能。
4. 玩家击杀 4 只教学怪物。
5. 教学靠近自动拾取、掉落光柱品质和首次掉落反馈。
6. 玩家返回陆青处，打开背包查看战利品。
7. 教学穿戴装备和服用药品。
8. 陆青告知玩家前往灵植园寻找周文扬丹师。

### 关键对话

**玩家：**“弟子见过陆师兄。”

**陆青：**

> 无需拘谨，往后你我便是同门。师兄赠你一枚储物袋当做见面礼，希望对你有所帮助。

**陆青：**

> 师弟初入宗门，我尚未知晓你的修为根底，可愿随我前往宗门幻境试炼一番，我好为你日后修行之路指点方向。

**玩家：**“晚辈愿往。”

**试炼结束后，陆青：**

> 经过刚刚的一番测试，我已大概知晓你的基本情况，我们先离开此地再谈。

**返回后，陆青：**

> 方才在幻境中见师弟也是略有所得，可以打开储物袋看看有哪些物品有助于你提升实力。

> 师弟已至练气七重境，境界虽有些不稳，但也可以通过服用丹药进行巩固和提升。稍后你前往灵植园，寻到周文扬丹师，你与他讲是陆青师兄让你来的，他便会知晓该如何助你。

### 设计意图

- 在低风险副本中完成第一次战斗闭环：选敌、攻击、击杀、拾取、使用战利品。
- 通过储物袋把背包解锁包装为世界内事件。
- 战斗结束后立即使用掉落，建立“战斗获得资源—资源提升角色”的认知。

### 所需支持与资源

- 试炼幻境副本、教学怪物、出生点与退出点。
- 目标选择、普攻、技能栏和战斗教学。
- 掉落表、自动拾取、品质光柱、音效和飘字。
- 背包解锁、装备穿戴、药品服用教学。
- 首次击杀及首次掉落埋点。

### 异常处理

- 教学怪物死亡后未拾取物品：离开副本前提醒，必要时自动补发。
- 玩家死亡：原地复活或重置当前波次，不清空教学进度。
- 背包已满：关键物品进入临时背包或邮件。

### 完成与验收

- 击杀计数达到 4、完成至少一次拾取及一次背包操作后允许交付。
- 未完成教学步骤时，出口和交付不得提前完成任务。

## 6.3 主线任务 1.3：丹道初窥——首次炼丹

### 基础信息

| 项目 | 内容 |
|---|---|
| 前置条件 | 完成任务 1.2 |
| 开始 NPC | 灵植园药童 |
| 结束 NPC | 周文扬 |
| 地点 | 灵植园、炼丹房 |
| 核心教学 | 采集、炼丹、丹药分类、丹毒、调息 |

### 玩家可见目标

1. 前往灵植园寻找周文扬。
2. 帮助药童采集凝露草。
3. 携带炼丹材料前往炼丹房。
4. 制作 3 颗引气丹。
5. 服用 1 颗引气丹。
6. 打坐调息约 10 秒。

### 流程与玩家操作

1. 玩家在灵植园遇见药童，询问周文扬的位置。
2. 药童要求采集成熟的凝露草作为交换。
3. 玩家完成采集，获得炼丹材料和路线指引。
4. 玩家在炼丹房找到周文扬并接受搭脉检查。
5. 点击炼丹炉，打开炼丹面板，制作 3 颗引气丹。
6. 系统说明丹药分类、每日服用上限与丹毒。
7. 玩家服用 1 颗引气丹，获得 `+50 修为`反馈。
8. 玩家打坐调息约 10 秒。
9. 周文扬指引玩家前往功法阁。

### 关键对话

**玩家对药童：**

> 这位师兄（姐），弟子刚拜入山门不久，冒昧打扰，不知师兄（姐）可知晓周丹师身在何处？是陆青师兄嘱托弟子前来拜见。

**药童：**

> 想找周丹师也行。方才园中活儿繁多，我一人忙不过来。你若是肯帮我把这几株成熟的凝露草尽数采摘干净，我便告诉你周丹师的所在。

**采集后，药童：**

> 倒是个勤快靠谱的新人。这些便是周丹师让我来采摘的，你带上这篮灵植，顺着这条路直行，便可到达炼丹房。

**周文扬：**

> 你内息紊乱，根基有些不稳。我正要炼制固本培元的丹药，与你有益，你来搭把手吧。

**炼丹后，周文扬：**“服下这枚引气丹，调理内息。”

**服用后，周文扬：**

> 嗯。不过，丹药虽好，每日不可多服，若贪多服用会积累毒性，反而有害，切记。

**玩家：**

> 弟子谨记。弟子还有一事想问，请问周师兄功法阁如何前往？莫黎长老正在功法阁等候弟子。

**周文扬：**“功法阁就在丹房北面，出门后过桥便是了。”

### 设计意图

- 建立采集、加工、使用的生产闭环。
- 让玩家理解丹药能加速成长，但受到每日上限和丹毒约束。
- 用“内息不稳”解释为什么需要炼丹与调息。

### 所需支持与资源

- 灵植采集点、成熟状态、采集动作和进度反馈。
- 炼丹炉交互、配方、材料消耗、产物发放。
- 丹药分类、每日上限、丹毒数据和提示。
- 使用丹药后的修为飘字及打坐进度条。
- 药童性别/称谓最终配置方案。

### 完成与验收

- 关键材料不可出售、丢弃或被其他配方消耗。
- 炼丹失败时不得阻断新手流程，建议首次炼丹必定成功。
- 服药和调息均完成后才可结束任务。

## 6.4 主线任务 1.4：长老授业——传授功法

### 基础信息

| 项目 | 内容 |
|---|---|
| 前置条件 | 完成任务 1.3 |
| 开始/结束 NPC | 莫黎长老 |
| 地点 | 功法阁、试炼幻境 |
| 核心教学 | 秘籍使用、功法激活、技能使用、熟练度 |

### 玩家可见目标

1. 前往功法阁拜见莫黎长老。
2. 使用秘籍学习基础功法。
3. 前往试炼幻境使用新功法。
4. 完成试炼并获得功法熟练度。
5. 前往大殿广场寻找陆青。

### 流程与玩家操作

1. 玩家在功法阁塔楼前与莫黎长老对话。
2. 获得基础秘籍，在背包中点击使用。
3. 激活对应功法技能并显示技能说明。
4. 玩家进入试炼幻境，使用新技能击杀教学怪物。
5. 完成后返回莫黎长老处，获得功法熟练度反馈。
6. 莫黎长老指引玩家前往大殿广场。

### 关键对话

**莫黎长老：**

> 老夫唤你来此处，主要想传授你本门的基础修行功法，这都是门内历代大能精心纂写的，基础却精湛，你要好好研习。

> 悟性不错，即已学会了，不如随我前往试炼幻境，实践一二。

**试炼后，莫黎长老：**

> 适应得很好，老夫甚是欣慰。你要记住，修行一道，贵在持之以恒。你要日日打磨、勤修不辍，方能牢固功法，日后才可攀登更高境界。

> 你现在且前往宗门大殿广场，你陆青师兄正在那里等你，去吧。

### 设计意图

- 让玩家在获得技能后立刻使用，避免只读说明却不形成操作记忆。
- 引出功法熟练度作为长期成长维度。
- 强化莫黎长老的授业导师身份。

### 所需支持与资源

- 秘籍道具、背包使用及功法解锁接口。
- 技能栏自动装配或明确装配教学。
- 试炼怪物针对新技能的教学配置。
- 功法熟练度获取与结算表现。

### 完成与验收

- 必须检测到秘籍使用成功及至少一次新技能有效命中。
- 秘籍误删或丢失时可自动补发。

## 6.5 主线任务 1.5：再遇陆青——修炼基础

### 基础信息

| 项目 | 内容 |
|---|---|
| 前置条件 | 完成任务 1.4 |
| 开始/结束 NPC | 陆青 |
| 地点 | 宗门大殿广场水池旁 |
| 核心教学 | 打坐修炼、修为增长、离线修炼、服饰装扮 |

### 玩家可见目标

1. 前往大殿广场寻找陆青。
2. 打坐修炼 30 秒。
3. 提升 1 级或获得规定修为。
4. 领取并穿戴宗门入门服饰。
5. 前往勤务堂。

### 流程与玩家操作

1. 玩家在水池旁找到陆青。
2. 点击“打坐”进入修炼状态。
3. 系统说明修炼期间不能战斗/移动、修为按秒结算、灵气浓度影响速度。
4. 玩家持续打坐 30 秒，暂定 `+5 修为/秒`，合计 `+150 修为`。
5. 提示离线修炼与上线结算规则。
6. 陆青赠送入门弟子服饰，玩家完成穿戴。
7. 陆青介绍勤务堂及委托任务。

### 关键对话

**陆青：**

> 修炼之人，根基最为重要。平时除却课业和历练修行，多数弟子都会选择来到这大殿广场打坐吐纳。此地灵气汇聚、气场安稳，最适合静心稳基、打磨修为。你如今刚入宗门，道基尚浅，正好借此机会静下心来打坐调息，一定对你的修为增进大有裨益。

**打坐后，陆青：**

> 这是本宗门的入门弟子服饰，穿上这身衣裳，往后你便是正式的天元宗弟子了。

> 师弟往后除了课业和修炼以外，亦可前往宗门的勤务堂接取各种委托任务。完成委托后，可获得丰厚的奖励，既能历练自身，又能获取资源。勤务堂就在宗门东北处，你现在可前往了解一二。

### 设计意图

- 教授挂机/离线成长，建立玩家长期回流预期。
- 用穿上宗门服饰完成身份转变，形成视觉奖励。
- 从个人修炼自然过渡到可重复任务系统。

### 所需支持与资源

- 打坐按钮、修炼状态、修为按秒结算。
- 灵气浓度及时间段修炼倍率配置。
- 离线修炼记录、上限、结算弹窗和防篡改校验。
- 宗门服饰道具、装扮界面和穿戴教学。

### 完成与验收

- 玩家中断打坐时保留已结算修为，但必须累计完成规定时长。
- 断线重连后修炼状态和累计时长正确。
- 服饰领取失败时可以补领。

## 6.6 主线任务 1.6：历练委托——首次接取委托

### 基础信息

| 项目 | 内容 |
|---|---|

| 开始/结束 NPC | 孙玲 |

1. 前往勤务堂。
2. 与孙玲交谈，了解任务类型。
3. 打开任务栏。
4. 接取并完成第一个悬赏任务。
5. 返回孙玲处交付。

### 流程与玩家操作

1. 玩家进入勤务堂，与执事孙玲对话。
2. 孙玲介绍宗门任务、悬赏任务和世界任务。
3. 玩家与任务栏交互，打开任务面板。
4. 系统高亮第一个悬赏任务：“清理徘徊在山门外的魔物”。
5. 玩家接取并完成任务。
6. 返回孙玲处交付，获得奖励和宗门贡献说明。

### 关键对话

**孙玲：**

> 我叫孙玲，是勤务堂的执事。师弟初次接触，就由我来给你介绍一下委托任务类型。委托任务分为三大类，分别为宗门任务、悬赏任务和世界任务。宗门任务通常是宗门建设任务；悬赏任务分为低阶、中阶和高阶；世界任务是机缘任务为主，可遇不可求。

**任务面板提示：**

> 悬赏任务每日刷新，是资源获取的重要来源。

**完成后，孙玲：**

> 完成委托任务能够积攒宗门贡献，宗门贡献能够助你在门内出人头地。即为本门弟子，切记以宗门为荣，为宗门的发展尽自己一份力量。

### 设计意图

- 教授游戏的任务分类与日常循环入口。
- 让玩家亲自完成一次接取—追踪—执行—交付闭环。
- 提前建立宗门贡献与资源成长的关系。

### 所需支持与资源

- 任务面板、分类页签、刷新时间和奖励预览。
- 任务栏场景交互及任务接取教学。
- 悬赏怪物、导航、计数和交付逻辑。
- 宗门贡献货币及获得反馈。

### 完成与验收

- 首个悬赏任务在新手阶段固定可用，不受随机刷新影响。
- 玩家已提前击杀目标时，明确是否追溯计数。
- 每日刷新不得导致新手固定任务消失。

## 6.7 主线任务 1.7：筑基之路——收集突破材料

### 基础信息

| 触发方式 | 委托面板自动刷新突破任务 |
| 引导 NPC | 孙玲 |

5. 准备灵石×100。
6. 返回孙玲处确认材料。

### 流程与玩家操作

1. 玩家达到练气圆满后自动出现世界任务/突破任务提示。
2. 孙玲说明筑基需要特殊历练和材料。
3. 任务面板展示材料来源并支持逐项追踪。
4. 玩家击杀魔化 BOSS，获得核心材料“魔气结晶”。
5. 玩家采集或提交副材料和灵石。
6. 材料齐集后返回孙玲处，获得前往渡劫台的指引。

### 关键对话

**孙玲：**

> 师弟的修为已至练气大圆满，若想突破练气、踏入筑基，需要经过特殊的历练。你可前往任务栏接取突破任务。

**材料齐集后，孙玲：**

> 师弟既然已经把材料集齐了，那便前往渡劫台吧，莫长老在那里候你，预祝师弟顺利突破。

### 设计意图

- 将境界突破塑造成需要准备和历练的重要事件。
- 教授多来源材料追踪：BOSS、采集、货币。
- 为最终渡劫建立期待和仪式感。

### 所需支持与资源

- 境界/修为条件监听及自动任务触发。
- 突破材料面板、来源跳转、数量追踪和提交。
- 魔化 BOSS、核心材料保底掉落。
- 关键材料保护和重复获取规则。

### 异常处理

- BOSS 未掉落核心材料：新手首次应保底掉落或任务直接发放。
- 玩家提前持有材料：建议计入任务进度，避免重复劳动。
- 消耗关键材料后任务无法继续：任务期间锁定或提供补救任务。

### 完成与验收

- 境界、核心材料、副材料和灵石全部满足后才可完成。
- 材料扣除时机需统一：建议在正式点击突破时扣除，而非孙玲确认时扣除。

## 6.8 主线任务 1.8：破境筑基——首次大境界突破

### 基础信息

| 项目 | 内容 |
|---|---|
| 前置条件 | 完成任务 1.7；突破材料齐集 |
| 开始/结束 NPC | 莫黎长老 |
| 陪同 NPC | 陆青、周文扬 |
| 地点 | 渡劫台、心魔秘境 |
| 核心教学 | 突破面板、心魔战斗、雷劫结算、境界晋升 |

### 玩家可见目标

1. 前往渡劫台。
2. 与莫黎长老交谈。
3. 打开修炼面板并点击“突破”。
4. 在心魔秘境击败镜像分身。
5. 承受雷劫并完成筑基。

### 流程与玩家操作

1. 玩家抵达渡劫台，看到莫黎长老、陆青和周文扬。
2. 对话后高亮修炼面板中的“突破”按钮。
3. 点击突破并扣除突破材料，进入心魔秘境。
4. 心魔以玩家镜像分身出现，美术表现为角色黑化。
5. 击败心魔后突破进度达到 50%。
6. 返回/切换到渡劫台，播放天气异象和 5 秒雷劫倒计时。
7. 首次筑基按当前方案默认 100% 成功。
8. 成功后播放全屏金光、世界播报和境界晋升反馈。
9. 与莫黎长老完成结尾对话，显示“通天大道第一章终了”。

### 关键对话

**莫黎长老：**“看来你已经准备好了。”

**玩家：**“弟子已做万全准备，只待时机。”

**突破成功后，莫黎长老：**

> 苦修有成，境界进阶，值得嘉许。修行之路漫漫长，突破不是结束，是新的开始，切莫心生自满，踏实砥砺方能大道长远。

**玩家：**“弟子时刻谨记。”

### 成功表现

- 全屏金光特效。
- 雷劫、环境光和天气恢复。
- 世界播报：“【玩家名】成功筑基，踏上仙途！”
- 境界变化、属性提升和新功能解锁提示。
- 屏幕大字：“通天大道第一章终了”。

### 设计意图

- 用战斗、演出和系统结算共同制造第一次境界突破高潮。
- 让玩家完整理解未来突破的基本结构：准备材料—挑战心魔—渡劫—结算。
- 以第一章结束提示给予明确阶段成就感。

### 所需支持与资源

- 修炼面板突破入口、条件检查和材料扣除。
- 玩家镜像生成、技能复制、数值缩放和黑化材质。
- 渡劫台天气、雷电、倒计时、全屏金光和世界播报。
- 突破成功/失败结算、属性刷新和存档。
- 陆青、周文扬陪同站位及环境演出。

### 异常处理

- 心魔战斗失败：允许原地重试，不重复扣除材料。
- 战斗中退出或断线：返回心魔阶段或突破前安全节点。
- 雷劫结算期间断线：结果由服务端确定，重连后补播结果。
- 不允许重复获得首次筑基奖励和世界首播表现。

### 完成与验收

- 成功后境界、属性、任务状态和奖励必须同时持久化。
- 镜像必须正确读取玩家外观和可用技能，不得复制无法处理的特殊技能。
- 首次筑基成功率和失败逻辑需在上线前最终确认。
