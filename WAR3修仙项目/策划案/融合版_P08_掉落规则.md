# V0 掉落规则（融合版）

> 版本：vMerge-1.1 ｜ 2026-06-09 ｜ 对应交付清单 P08 ｜ 对应程序 S12
> 融合来源：小马《V0_掉落规则》(主体,原子拾取/防作弊/障碍回缩/闪烁渐隐最完整) + 小龙《V0 掉落规则》(对比项)。
> **归属裁决（总分析）：最后一击 / 保护 5s / 拾取 200 码（取小马）。** 小龙的"首次伤害/10s/150码"是多人防抢机制，V0 灰盒单人为主用不上，已否决。

## 一、掉落触发

| 规则 | 定义 |
|------|------|
| 触发时机 | 怪物 hp≤0 当帧（死亡瞬间），死亡动画播放时 0.3s 内触发 |
| 触发次数 | 每怪一次 |
| 掉落来源 | 读 monster_config 的 drop_scheme_id → 查 drop_config（P09） |
| 事件驱动 | S12 监听 S08 死亡事件，不在战斗代码里直接调掉落函数（不硬耦合） |

## 二、权重池随机（32 槽）

```
1. 收集 weight>0 的槽，total_weight = Σweight
2. r = random(1, total_weight)（用 GetRandomInt，勿用 rand()%total 防模偏差）
3. 遍历累加 weight，r≤累计值 → 命中该槽
4. 数量 = random(count_min, count_max)
5. force_drop=true 的槽额外独立产出（V0 无此场景，全 false）
6. 本次掉落种类数 N = random(scheme.min_drops, scheme.max_drops)，重复抽 N 次
```

## 三、归属判定【裁决落地】

| 规则 | 定义 |
|------|------|
| 归属者 | **最后一击**击杀怪物的玩家（谁打死归谁） |
| 保护期 | **5 秒**（drop_claim_time），从掉落物生成起计 |
| 保护期内 | 仅归属者可拾，其他玩家拾取提示"该物品归属他人" |
| 保护期后 | 任何人可拾 |
| 归属者断线 | 物品不受影响，继续存在至 60s；5s 后任何人可拾 |
| 无玩家击杀（环境伤害） | 无归属者，直接公开 |
| 归属时间 | 用**服务器时间戳**，防客户端改时间作弊 |

> V0 不做：队伍/公会共享、DPS 分配、轮流拾取、持有权转移。

## 四、掉落位置

```
单物品：坐标 = 怪物死亡坐标（偏移 0）
多物品：坐标 = 死亡坐标 ± 随机偏移（X/Y ∈ [-64,64] 码），半径约 64 码圆内散落
  → 每件独立随机，与已有地面物品距离<32 码则再偏移一次
障碍检测：目标点不可通行 → 向死亡点回缩 32 码，最多 3 次；仍不行 → 落死亡坐标（必可通行）
```

V0 灰盒为平地，障碍逻辑大概率不触发，但代码必须实现。

## 五、地面表现与消失

| 元素 | 规格 |
|------|------|
| 图标 | ground_model（空则复用 icon_id 缩小至 24×24），2D 面片始终面向镜头 |
| 悬浮文字 | 物品名，白色 12px，图标上方 (0,+20) |
| 选中高亮 | 鼠标悬停图标边缘白色轮廓 1px，文字加粗 |
| 存在时间 | 60 秒 |
| 闪烁 | 50-55s 开始闪烁（0.5s/次），55-60s 加速闪烁（0.25s/次）+ alpha 渐降至 0 |
| 超时 | 60s 未拾 → 销毁，不进背包、不补偿、不记日志；向归属者提示"掉落物已消失" |
| 拾取后 | 立即消失 → 数据转入背包 → 服务器确认后广播 |

> V0 不做：品质颜色/光柱/3D 模型/弹射爆散/旋转/阴影。

## 六、防重复拾取（原子操作）

```
玩家点击地面物品 / 按 F 键（拾取范围 200 码）
  → 客户端 PICKUP_REQUEST(item_id, player_id)
  → 服务器原子操作（同一锁内完成 4 步校验）：
      ① 物品仍在地面？
      ② flag_picked == false？
      ③ 归属通过？（保护期内=归属者 / 保护期后=任何人）
      ④ 背包有空间？（可堆叠 / 有空槽）
  → 全过：flag_picked=true（先标记防重复）→ 移出地面列表 → 入包 → 广播消失 → 回 PICKUP_SUCCESS
  → 任一失败：回失败原因 + 物品保持地面（拾取失败不消耗，倒计时不变）
```

服务器端是地面物品的唯一权威，客户端只有渲染镜像，所有拾取在服务器校验。

## 七、配置表字段

### drop_global_config（全局参数）

| 字段 | 建议值 | 说明 |
|------|--------|------|
| drop_lifetime | 60.0 | 地面存在总时长(秒) |
| drop_claim_time | 5.0 | **归属保护时长(秒)** |
| drop_blink_start | 50.0 | 开始闪烁(秒) |
| drop_fade_start | 55.0 | 开始渐隐(秒) |
| pickup_range | 200.0 | **拾取范围(码)** |
| scatter_radius | 64.0 | 多物品散落半径(码) |
| pickup_timeout | 2.0 | 客户端拾取请求超时(秒) |

### 怪物关联字段

monster_config.`drop_scheme_id`（int，0=不掉落）→ 引用 drop_config.scheme_id（P09）。
