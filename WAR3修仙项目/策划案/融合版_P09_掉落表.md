## 一、掉落机制

`drop_type` = "掉落方式_按概率"：**32 槽独立掷骰**。每个槽独立判定（详见 P08 §二），命中则产出该槽配置的数量份物品。

---

## 二、配置表字段（drop_config）

### 方案级字段

| # | 字段名 | 类型 | 说明 |
|---|--------|------|------|
| 1 | `id` | int | 方案唯一 ID（公式生成），主键 |
| 2 | `drop_type` | enum | 掉落方式："掉落方式_按概率" |
| 3 | `time_limit_id` | int | 掉落时间限制 ID |
| 4 | `min_drop_number` | int | 最小掉落数 |
| 5 | `max_drop_number` | int | 最大掉落数 |
| 6 | `drop_number` | int | 掉落物数量（公式计算） |
| 7 | `drop_items` | array | 32 槽掉落物数组，每槽结构见下 |

### 槽位子字段（每槽 8 列，共 32 槽）

| 子字段 | 类型 | 说明 |
|--------|------|------|
| 掉落物ID | int | 物品 ID |
| 掉落物名称 | string | 物品名称 |
| main_class | string | 物品大类 |
| sub_class | string | 物品子类 |
| lv | int | 物品档次 |
| prob | int | 掉落概率，单位 1/1e8（如 100000000 = 100%） |
| num | int | 命中后产出数量 |
| tips_prob | int | 系统广播概率 |
