## 三、角色名输入框（六状态）

| 状态 | 视觉 | 触发 |
|------|------|------|
| 默认 | 灰边框，placeholder"请输入角色名" | 初始 |
| 聚焦 | 蓝边框，光标闪烁 | 点击/Tab |
| 通过 | 绿边框/恢复默认 | 校验通过 |
| 失败 | 红边框 + 下方红字错误提示 | 校验失败 |
| 提交中 | 输入框锁定，按钮转圈 | 点确认后 |
| 成功 | 短暂绿勾 | 建档成功 |

校验规则（P13 锁定）：min=1，max 8 中文/16 英文，禁纯数字/纯空格/特殊符号，V0 敏感词关。错误文案见 P13/P24。

## 四、灵根选择区

- 5 个灵根块（金◇/木♧/水💧/火♨/土▣），数据来自 linggen_config（P15）。
- 默认全未选；点击选中（边框+发光，文字变五行色+加粗）；5 选 1 互斥。
- 悬停/选中显示 Tooltip（灵根名+一句话+释义，来自 P15 tooltip 字段）。
- 区域底部强制显示灰字「灵根效果将在后续版本开放」。

| 3 | 水灵根 | 💧 | #3399FF |

## 五、确认按钮

- 文字"踏入修仙"。
- 灰显条件：角色名未通过校验 或 未选灵根。两者都满足才可点。
- 点击 → 提交中状态 → 调 S05 建档 → 成功进入游戏 / 失败回显错误（P13 错误码）。

---

## 六、快捷键与无障碍

| 键 | 行为 |
|----|------|
| Tab | 焦点在 输入框 ↔ 灵根区 ↔ 确认按钮 切换 |
| 1-5 | 快速选金/木/水/火/土灵根 |
| Enter | 等同点"踏入修仙"（条件满足时） |
| Esc | V0 不可取消（强制创角），无响应或提示 |

无障碍：错误提示文字+边框双重反馈（不仅靠颜色）；灵根选中态除颜色外有边框/发光区分（色盲可辨）。
