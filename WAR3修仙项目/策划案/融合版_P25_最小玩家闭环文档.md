## 一、一句话闭环

**进房 → 建角 → 入场 → 打怪 → 掉落 → 拾取 → 背包 → 保存 → 重进读取，全部数据一致。**

---

## 二、玩家视角导览（首次体验）

> 此节为体验导览，技术细节见后文。

1. **进房**：WC3 → KK 平台 → 找修仙地图 → 创建/加入房间 → 房主"开始"→ 加载(<30s)。
2. **建角**：弹创角面板 → 输入道号（1-8 汉字）→ 选灵根（金木水火土五选一）→ 见"灵根效果将在后续版本开放"→ 点"踏入修仙"。
3. **入场**：角色出现在安全区出生点（白圈）。HUD：道号+"练气一层"、红血条 100/100、蓝灵力条 50/50、灰修为条(V2开放)。背包有聚气丹×5+灵泉水×3，灵石 100。
4. **打怪**：向东出安全区进刷怪区 → 走近"小野狼" → 选中 → A 键攻击 → 白字"8" → 小野狼还击头顶飘"1" → 约 4 下击杀 → 死亡动画 → 掉落物出现。
5. **拾取**：走近(200码内)点击掉落物 → "获得 灵石 ×2" → 入背包。
6. **背包**：B 键开背包 → 见灵石×2 → 再打再得灵石 → 同格变"灵石 ×3"。
7. **用药**：打石傀儡掉血 → 右键聚气丹 → 绿字"+50" → 血回升 → 数量-1。
8. **装备**：击杀练气妖兽掉铁剑 → 右键穿戴 → "已装备 铁剑" → 攻击 10→13 → 武器位显示。
9. **存档退出**：自动存档闪过"道基已固（保存成功）"→ 退出。
10. **重进**：再进 → 不弹创角 → 道号/境界/属性/背包/铁剑/灵根全部与退出前一致。

```

  进房→建角(命名+选灵根)→入场(安全区出生)
```
