# WC3修仙项目 · 遗留问题

> 本文记录世界观/系统设计中已识别但尚未解决的逻辑缺口和待补机制。
> 与 `待决.md`（thoth 全局待决）的区别：本文是 WC3修仙项目专属的技术性遗留，面向策划/程序。

---

## 一、严重：阻塞终局设计

### 1. 归渊机制缺失

**问题**：魔族升格机制只有命名（"归渊"），无物质法则层面的推导链。

**现状**：第一因 §六写"暂以归渊为最佳候选——高阶魔族摆脱怨恨，回归原初混沌。但不排除入魔渊/逆升格等方向。"但没有回答：物理上发生了什么？魔气结构如何重组？与魔核的关系是什么？

**影响**：V6/V7 终局涉及魔界深层内容时，魔族的行为逻辑缺少底层规则支撑。

**对比**：灵魂三步有完整推导链（涌现→连续→自举），丹毒有三层模型（神识/佐药/丹田），归渊只有名字。

**方向提示**（非决定）：
- 魔核结构重组：被封印怨化的魔气解压→重新连通原初混沌？
- 或者：归渊不是"恢复"，是"第二次涌现"——魔族在怨恨中自举出的"我们"主动解散？

---

### 2. 飞升后个体/集体比例悬置

**问题**：飞升者接入天界蚁群后，个体与集体的具体比例、自由度、同化速度全部悬置。

**现状**：第一因 §四写"飞升后个体与集体的具体比例，暂时悬置。但方向明确：飞升是个体与整体并存，而不是单纯成神，也不是单纯消失。"

**影响**：玩家最终飞升时的体验不可设计——是变成可控的新形态（新玩法），还是变成 NPC（叙事收束），还是软性的"你还在但被拉扯"（渐进式）？三种方向对应完全不同的终局设计。

**方向提示**（非决定）：
- 个体主导：飞升者保持高度自主，蚁群是"可选接入的集体智慧"
- 集体主导：飞升者逐渐融入，个体边界模糊（"被同化"是渐变而非突变）
- 动态平衡：在不同场景下比例可调——日常个体主导，重大决策集体涌现

## 二、中等：逻辑需澄清

### 3. 器灵与兽魂的边界矛盾

**问题**：物质法则说"兽魂可以作为器灵前身（种子）"，但第一因定义灵魂是"自己产生自己"的自举过程。如果兽魂只是"意识在肉体死亡后短暂的残留"，它有多少自举性？炼入法宝后长成的器灵是原兽魂的延续，还是一次全新的涌现？

**现状**：物质法则 §十二·器灵诞生的描述与第一因 §二·灵魂三步之间没有衔接。

**建议方案**（待确认）：
> 兽魂作为种子提供的是涌现的初始条件（方向偏好、气质底色），最终器灵是法宝内部秩序结构复杂到临界点后自身涌现的结果，非原兽魂的直接延续。类似"用一首诗的第一行启动了一个新诗人的创作"——起笔有来处，诗是新的。

## 三、轻微：待补/维护

### 4. 五行生克战斗规则未展开

**问题**：物质法则说了"五行生克不是绝对胜负——境界、灵气密度、功法理解都能压过"，但没有同层级下的修正公式或交互规则。

**影响**：战斗系统如要调用五行，需要数值接口。当前只有定性描述。

**建议**：进战斗系统详细设计时补，不在世界观文档中展开。
