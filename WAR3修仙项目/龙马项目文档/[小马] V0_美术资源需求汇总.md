## 三、A01 灰盒地图

### 规格
- 尺寸：128×128（WC3 中等）
- 地形：**纯平地**，无起伏/水域/装饰
- 纹理：WC3 默认，不做修仙风格
- 光照：WC3 默认白天
- 加载：<30s

### 6 个功能区

| 区域 | 地面 | 标记色 | 说明 |
|------|------|------|------|
| 出生区 (0,0) | 浅色草地 | 白圈(半径64)+光柱+"出生点"文字 | 嵌套在安全区内，出生/复活点 |
| 安全区 | 浅色草地/石板 | 蓝色边框 | 无怪 |
| 刷怪区A (800,0) | 暗色草地 | 黄圈+"小野狼"+"试炼区" | 小野狼×3+石傀儡×2 |
| 刷怪区B (1600,0) | 暗色草地 | 橙圈+"练气妖兽"+"危险区" | 练气妖兽×1 |
| 掉落测试区 (0,-700) | 沙地 | 无特殊标记 | 空地，人工放怪 |
| 边界 | 黑/悬崖 | 不可通行 | 外围悬崖+路径阻断器 |

### 路径
- 路径1：出生→刷怪A，正东 ~800×200，浅色纹理带
- 路径2：刷怪A→刷怪B，正东 ~800×200（含过渡走廊）

### 标记优先级
```
P0 必须：出生点标记 / 刷怪区A黄圈+文字 / 刷怪区B橙圈+文字 / 边界悬崖阻挡
P1 建议：安全区蓝框 / 路径纹理 / 区域浮空文字("试炼区""危险区")
P2 锦上：安全区浅色地面 / 掉落测试区沙地 / 小地图轮廓
```

### 颜色约定
白=出生/复活，蓝=安全，黄=低难度试炼，橙=中难度危险

## 四、A02 创角界面

### 面板
- 尺寸：**500×600px**，居中
- 风格：WC3 原生黑底灰框（深灰半透明底 rgba(0,0,0,0.6)，浅灰边框 2px）
- 标题："创建角色"（白色 18px，顶部居中）
- 模态阻塞（不创角不进游戏）

### 布局（自上而下）
```
┌──────────────────────┐
│      创建角色         │  标题
├──────────────────────┤
│ 角色名 [__________]   │  输入框 + 规则提示
│ └ 规则提示/错误(红)   │
├──────────────────────┤
│ 选择灵根              │
│ [◇金][♧木][💧水][♨火][▣土] │  5个灵根块
│ └ 灵根简介(悬停显示)   │
│ 「灵根效果将在后续版本开放」 │  灰色强制标注
├──────────────────────┤
│     [ 踏入修仙 ]       │  确认按钮
└──────────────────────┘
```

### 需要的美术素材

| 素材 | 规格 |
|------|------|
| 面板底框 | 500×600，黑底灰框 |
| 输入框六状态 | 默认(灰框)/聚焦(蓝框)/通过(绿框)/失败(红框+红字)/提交中(锁定+转圈)/成功(绿勾) |
| 灵根块选中态 | 边框发光 + 文字变五行色+加粗 |
| 灵根块未选态 | 默认灰框 |
| "踏入修仙"按钮可点态 | 正常 |
| "踏入修仙"按钮灰显态 | 灰显不可点 |
| 强制标注文字 | 灰色 "灵根效果将在后续版本开放" |
| 缩放适配 | 各分辨率不错位 |
