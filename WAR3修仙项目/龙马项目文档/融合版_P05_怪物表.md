### 2.1 与玩家的战斗模拟（玩家练气一层 hp100/atk10/def5）

| 场景 | 公式 | 单次伤害 | 击杀次数 | 耗时(2.2s/次) |
|------|------|---------|---------|--------------|
| 玩家→小野狼 | max(10-2,1) | 8 | 4 | ~9s |
| 小野狼→玩家 | max(6-5,1) | 1(保底) | 100 | — |
| 玩家→石傀儡 | max(10-4,1) | 6 | 10 | ~22s |
| 石傀儡→玩家 | max(8-5,1) | 3 | 34 | ~75s |
| 玩家→练气妖兽 | max(10-3,1) | 7 | 12 | ~26s |
| 练气妖兽→玩家 | max(11-5,1) | 6 | 17 | ~37s |

- 小野狼：4 刀杀，对玩家仅 1 点保底伤害 → 无威胁，验证战斗链跑通。
- 石傀儡：22 秒杀，34 次攻击可致死 → 低威胁，需关注血量。
- 练气妖兽：玩家 26s 杀它 / 它 37s 杀玩家 → 略占优但必须主动进攻，验证死亡→复活闭环。

## 三、字段说明（关键项）

- `monster_id`：主键，范围 1001-1999（怪物 ID 段）。≠ 运行时 unit_id。
- `level`：仅标注难度梯度，**V0 不参与任何公式**。
- `atk`/`def`：单值，参与 `max(atk-def,1)`。**严禁旧传奇上下限/物防魔防结构。**
- `atk_speed`：攻击间隔（秒）。**字段预留差异化**，V0 三怪暂统一 1.5（= combat_config 默认攻击冷却，保证与 P03 战斗节奏自洽）。后续版本可逐怪差异化。
- `chase_range`：显式追击范围（码），值 = aggro_range × 1.5（与 P06 AI 脱战逻辑一致）。超出此范围持续 3s 脱战归位。
- `model_id`：V0 用 WC3 自带中立模型（Wolf/Golem/Wendigo），动画完整。V1+ 换修仙自定义模型。
- `drop_scheme_id`：关联 P09 掉落表 scheme_id（1/2/3）。

## 四、配置表字段定义（monster_config）

| 字段名 | 类型 | 必填 | 默认 | 说明 |
|--------|------|:--:|------|------|
| `monster_id` | int | ✅ | — | 怪物唯一 ID（1001-1999），主键 |
| `monster_name` | string | ✅ | — | 2-6 中文字符 |
| `level` | int | ❌ | 1 | 难度标注，不入公式 |
| `hp` | int | ✅ | — | 生命值 |
| `mp` | int | ✅ | 0 | V0 怪物固定 0 |
| `atk` | int | ✅ | — | 攻击（单值） |
| `def` | int | ✅ | — | 防御（单值） |
| `move_speed` | int | ❌ | 150 | 码/秒 |
| `atk_range` | int | ❌ | 128 | 攻击范围（码） |
| `atk_speed` | float | ❌ | 1.5 | 攻击间隔（秒），不填用 combat_config 默认 |
| `aggro_range` | int | ❌ | 400 | 索敌范围（码） |
| `chase_range` | int | ❌ | aggro×1.5 | 追击范围（码），不填则按 aggro_range×1.5 计算 |
| `model_id` | string | ✅ | — | WC3 模型代码 |
| `model_scale` | float | ❌ | 1.0 | 缩放 0.5-2.0 |
| `respawn_time` | int | ❌ | 30 | 重刷间隔（秒） |
| `drop_scheme_id` | int | ❌ | 0 | 掉落方案 ID，0=不掉落 |

## 五、融合说明（本份是真·双版合并，记下取舍）

| 维度 | 取自 | 理由 |
|------|------|------|
| 命名 + monster_id(1001-1003) | 小马 | G1：练气妖兽锚定境界词汇；分段编码与物品同构 |
| 数值(hp/atk/def/move/aggro/respawn) | 小马 | 与 P03 §7.1 战斗模拟互校，梯度更紧（4/10/12 刀），练气妖兽可杀玩家验复活 |
| model_id/model_scale | 小马 | 小龙用 nfbr/ngol/nwlv 同为 WC3 占位，等价；取小马 |
| **level 字段** | 小龙 | 难度标注，无害，保留 |
| **atk_speed 字段** | 小龙(字段) | 为差异化预留；V0 值统一 1.5 以与 P03 节奏自洽（克制） |
| **chase_range 字段** | 小龙(字段) | 显式存表比小马"派生 aggro×1.5"更可配；值仍按 aggro×1.5 与 P06 一致 |
| CSV/JSON/字段详释/战斗模拟 | 小马 | 完整度更高 |
