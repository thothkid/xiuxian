## 五、配置表字段定义（drop_config）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `scheme_id` | int | ✅ | 方案唯一 ID（1-3），主键 |
| `scheme_name` | string | ✅ | 方案名 |
| `drop_way` | enum | ✅ | V0 固定 `mixed`（多槽权重池） |
| `min_drops` | int | ✅ | 每次最小掉落种类数 |
| `max_drops` | int | ✅ | 每次最大掉落种类数 |
| `slots` | JSON | ✅ | 32 槽数组，每槽 `{item_id, weight, count_min, count_max, force_drop}`，weight=0 不参与随机 |
