## 一、配置表字段（item_config）

| # | 字段名 | 类型 | 说明 |
|---|--------|------|------|
| 1 | `id` | int | 物品唯一 ID，主键 |
| 2 | `name` | string | 物品名称 |
| 3 | `quality` | enum | 品质：白色/绿色/蓝色/紫色/橙色/红色 |
| 4 | `dur_t` | int | 存在时间（地面消失） |
| 5 | `main_class` | string | 大类 |
| 6 | `sub_class` | string | 子类 |
| 7 | `drop_level` | int | 掉落档次 |
| 8 | `display_type` | enum | 显示分类 |
| 9 | `attrLimit` | int | 属性限制 |
| 10 | `keep_dur_t` | int | 有效时间 |
| 11 | `is_drop` | bool | 是否可丢弃 |
| 12 | `is_auto_use` | bool | 是否自动使用 |
| 13 | `required_level` | int | 使用级别下限 |
| 14 | `cultivation_tag` | string | 修炼标签 |
| 15 | `pile_max` | int | 堆叠上限 |
| 16 | `function_type` | string | 功能类型 |
| 17 | `function_id` | int | 功能关联 ID |
| 18 | `function_value` | int | 功能数值 |
| 19 | `desc` | string | 物品说明 |
| 20 | `spec_desc` | string | 获得说明 |
| 21 | `icon_id` | string | 背包图标 |
| 22 | `mdx_id` | string | 地表模型 |
| 23 | `light_eff_id` | string | 掉落光柱模型 |
| 24 | `border1` | string | 静态边框 |
| 25 | `border2` | string | 动态边框 |
| 26 | `tags` | string | 标签 |
