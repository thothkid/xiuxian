# V0 灰盒区域表（融合版）

> 版本：vMerge-1.1 ｜ 2026-06-09 ｜ 对应交付清单 P18 ｜ 对应程序 S06 / 美术 A01
> 融合来源：小马《V0_灰盒区域表》(框架/嵌套出生区/RegionManager 伪代码主体) + 小龙《V0 灰盒区域表》(并入掉落测试区)。
> 定位：V0 灰盒地图 6 个功能区的 ConfigRect 坐标。表名 `region_config`（G3）。按 G4：小马框架(0-4) + 小龙掉落测试区(5)。

---

## 一、地图总则

128×128（WC3 中等）平坦地形，平地为主、无起伏/水域/装饰。地图 = 平地 + 区域标记 + 路径纹理 + 边界阻挡。能跑、能区分区域即可。

区域包含关系：
```
出生区(0) ⊂ 安全区(1)
安全区 ⊥ 刷怪区A(2) ⊥ 刷怪区B(3)（水平相邻不重叠）
掉落测试区(5) 位于安全区南侧空地
边界区(4) = 包围全部的不可通行环带
```

---

## 二、区域定义（6 区）

| region_id | 区域名 | region_type | 坐标 (left,top,right,bottom) | 中心 | 说明 |
|:--:|------|------|------|------|------|
| 0 | 出生区 | safe | (-200, 200, 200, -200) | (0,0) | 出生/复活点，无怪 |
| 1 | 安全区 | safe | (-400, 400, 400, -400) | (0,0) | 外围安全区，无怪刷新 |
| 2 | 刷怪区A | spawn | (400, 400, 1200, -400) | (800,0) | 小野狼×3 + 石傀儡×2 |
| 3 | 刷怪区B | spawn | (1400, 400, 1800, -400) | (1600,0) | 练气妖兽×1 |
| 4 | 边界区 | boundary | 地图四边外缘环带 | — | 不可通行 |
| 5 | 掉落测试区 | loot_test | (-400, -500, 400, -900) | (0,-700) | 空地，人工放怪测拾取/背包 |

坐标系：WC3 世界坐标，X 正东、Y 正北。ConfigRect 四值：left=左X(最小)，top=上Y(最大)，right=右X(最大)，bottom=下Y(最小)。

---

## 三、区域判定

```
GetRegionId(x, y):
  优先级从内到外（出生区先于安全区，因嵌套）：
  for region in [0(出生),5(掉落测试),2(刷怪A),3(刷怪B),1(安全),4(边界)]:
    if left≤x≤right and bottom≤y≤top: return region_id
  return 4(边界/未定义)
```
出生区在安全区内部，判定时先查出生区命中再回落安全区。

## 四、配置表字段定义（region_config）

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|:--:|------|
| `region_id` | int | ✅ | 区域 ID（0 起），主键 |
| `region_name` | string | ✅ | 中文名 |
| `region_type` | enum | ✅ | safe/spawn/loot_test/boundary |
| `left` | float | ✅ | 左边界 X |
| `top` | float | ✅ | 上边界 Y |
| `right` | float | ✅ | 右边界 X |
| `bottom` | float | ✅ | 下边界 Y |

### CSV 示例
```csv
region_id,region_name,region_type,left,top,right,bottom
0,出生区,safe,-200,200,200,-200
1,安全区,safe,-400,400,400,-400
2,刷怪区A,spawn,400,400,1200,-400
3,刷怪区B,spawn,1400,400,1800,-400
4,边界区,boundary,-9999,9999,9999,-9999
5,掉落测试区,loot_test,-400,-500,400,-900
```
> 边界区为环带，程序按"在地图内但不在 0/1/2/3/5 任一区"判定，CSV 占位值仅标识。

## 五、边界实现（三方案，推荐悬崖）

| 方案 | 实现 | 推荐 |
|------|------|:--:|
| 悬崖边界 | 可玩区外抬升一圈悬崖 + 路径阻断器 | ⭐⭐⭐ |
| 树木边界 | 密集不可通行树木（间距<角色碰撞半径） | ⭐⭐ |
| 空气墙 | Pathing Blocker + 地面黑边 | ⭐ |

---

## 六、程序实现要点

1. region_config 加载到 RegionManager，提供 GetRegionId / IsInRegion / GetRegionCenter。
2. P07 刷新读 spawn_config 自带 center/radius（不再依赖 region 矩形随机），region_id 仅作归属/脱战边界。
3. 出生区嵌套安全区，判定按内层优先。
4. 边界用 Pathing Blocker 物理阻挡，不靠坐标判定踢回。
