## 一、面板总则

| 项目 | 规格 |
|------|------|
| 尺寸 | 350×400px（宽×高） |
| 位置 | **屏幕居中偏右**（不遮挡角色） |
| 风格 | WC3 原生黑底灰框（深灰半透明底，浅灰边框 2px） |
| 标题 | 顶部居中"背包"（白色 16px）+ 右上角关闭按钮 [×] |
| 快捷键 | **B** 开/关 |
| 关闭 | 再按 B / 点 [×] / 按 Esc |
| 模态 | **不阻塞**——打开时仍可移动/战斗/拾取 |
| 层级 | 在 HUD 之上，可与角色面板(C键)并存且互不遮挡关键信息 |

### V0 不做（UI 层）
物品拖拽交换 / 整理排序 / 拆分 / 装备对比双栏 / **丢弃**(右键无此项) / 底部快捷物品栏 / 背包扩容 UI / 分类标签页 / 物品锁定——全部 V1-V4。

## 三、物品槽

| 项目 | 规格 |
|------|------|
| 数量 | 6 格（可配 12），2 行 × 3 列 |
| 尺寸 / 间距 | 48×48px，间距 8px |
| 图标 | 居中 40×40px，来自 item_config.icon_id |
| 堆叠角标 | 数量 >1 时右下角白色数字（如"×3"）；装备(stack=1)不显角标 |
| **空格** | 深灰底 `#333333` + **虚线边框**（小龙，醒目区分空/占） |
| 左键 | 选中（高亮边框 `#999999`，比常态稍亮） |
| 右键 | 弹出菜单（见五） |

## 四、Tooltip（悬停，小马完整规格）

### 触发与位置
- 悬停 ≥ **0.3 秒**触发，移出立即消失；空槽不显。
- 默认位置：鼠标右侧 +10px 偏移。
- **右边界检测**：超出屏幕右侧 → 翻转到鼠标左侧。
- **下边界检测**：超出屏幕底部 → 向上偏移保持不溢出。
- 层级：高于背包面板，低于系统弹窗。

### 样式
背景 `rgba(20,20,20,0.92)`，边框 1px `#555555`，内边距 8px/10px，最小宽 120px / 最大宽 220px（超出换行）。物品名 14px 白色加粗；描述 12px 浅灰 `#CCCCCC`；行距 4px。

### 内容结构
```
┌──────────────────────────┐
│  物品名                   │  ← 14px 白加粗
│  物品描述                 │  ← 12px 浅灰 #CCCCCC（来自 item_config.description）
│  穿戴属性：攻击+3          │  ← 12px 绿 #66FF66（仅装备）
│  （当前：10 → 13）        │  ← 属性变化预览
└──────────────────────────┘
```
- 消耗品：恢复效果（聚气丹"生命+50"红 `#FF6666` / 回灵丹"灵力+50"蓝 `#66CCFF` / 灵泉水两行）。
- 灵石：显示背包中灵石总数（跨格统计）。
- 单栏，不做装备对比双栏（V3）。

## 五、右键菜单（按类型，与 P11 一致）

| 物品类型 | 菜单项 | 行为 |
|---------|--------|------|
| CONSUMABLE | 使用 | 恢复 hp/mp（P10），数量-1，HUD 血/灵条同步刷新 |
| WEAPON/ARMOR(背包) | 穿戴 | 入装备位，属性+（P12） |
| WEAPON/ARMOR(装备位) | 卸下 | 回背包，属性-（P12） |
| MATERIAL | （无操作） | 仅查看 |
| CURRENCY | — | 不进背包无菜单 |

菜单样式：深色背景、白字、悬停高亮，与系统弹窗一致。**丢弃 V0 不做**（右键菜单不出现该项）。

## 六、装备位显示

| 项目 | 规格 |
|------|------|
| 位置 | 物品槽下方，分隔线之下；标签"装备位"白 12px |
| 槽 | 武器位、衣服位，各 48×48px，水平排列间距 16px |
| **空位** | 深灰底 `#333333` + 浅灰边框 `#666666` + 居中灰字"空"（12px `#888888`） |

| 右键已装备 | 弹"卸下" → 卸回背包、属性恢复；背包满则提示"储物袋已满，无法卸下"，不变更 |
| 右键空位 | 不弹菜单 |
| 悬停已装备 | 同物品 Tooltip + 末行操作提示"右键卸下" |

## 七、交互细节

### 拾取/使用/穿脱后即时反馈
- 使用消耗品：角标-1，归零则格子清空（恢复空槽虚线深灰），HUD 血/灵条同步刷新。
- 穿戴：物品从背包格消失 → 装备位图标+名更新，角色面板属性刷新。
- 卸下：装备位清空("空") → 背包对应空格出现图标，角色面板属性刷新。
- 失败（满包/满血/已死）：弹系统提示文字（P24），UI 不变。

### 面板刷新时机（拾取成功/使用/穿戴/卸下后立即刷新）
刷新格子图标 + 堆叠角标 + 装备位 + 货币 + Tooltip 数据。

### 满包提示（走 P24，屏中上 2s 渐隐）
拾取满包"储物袋已满（背包已满）"；替换满包"储物袋已满，无法替换装备"；卸下满包"储物袋已满，无法卸下装备"。

### 拖拽（V0 不做，预留接口）
格子组件预留 `OnDragStart`/`OnDrop`，V0 留空。V3 做拖拽交换/拖拽穿戴。

### 与其他面板关系
HUD 始终显示不受影响；可与角色面板(C键)同开、不重叠遮挡关键信息；背包打开时左键空地仍走移动/攻击、右键背包物品走使用/穿戴（不冲突）；F 键拾取正常刷新。

## 九、红线（小马，务必遵守）
- **属性名只用 atk/def/hp/mp**：Tooltip 属性预览、装备位显示等所有 UI 文本严禁出现 力量/敏捷/智力。
- **图标只用 V0 icon_id**，不用老项目图标 ID。
- **V0 不销毁装备**：UI 上卸下/替换失败绝不导致装备丢失。
- **面板非模态**：打开背包不暂停/阻塞游戏。
