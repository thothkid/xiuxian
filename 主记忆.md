## 写入规则

1. 每条必须带状态标签：`[已确认]` / `[暂定]` / `[待验证]` / `[已废弃]`
2. 每条必须带日期
3. 每条必须明确属于哪一类：事实 / 判断 / 假设 / 决定
4. 不写流水账、不写过程、不写临时想法
5. 写入前须用户明确确认（说"写进主记忆"/"这条记下来"/"通过"等）
